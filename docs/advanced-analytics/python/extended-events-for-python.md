# <a name="extended-events-for-python"></a>Python 的擴充的事件

本節尚在開發中。

> [!NOTE]
> Python 支援是 SQL Server 2017 中的新功能，而且只在發行前版本。 尋找推出的詳細資訊。

一般情況下，您可以監視任何外部指令碼，包括執行 Python 的其中一個，請使用相同的架構，提供 SQL Server 2016 中執行 R 指令碼。 如需詳細資訊，請參閱[R 服務的擴充事件](../r/extended-events-for-sql-server-r-services.md)。

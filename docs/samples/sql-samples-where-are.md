## <a name="sql-samples-repository"></a>SQL 範例儲存機制

適用於 SQL 產品程式碼範例會在[Microsoft SQL Server Samples](https://github.com/microsoft/sql-server-samples) GitHub 儲存機制。 每個範例包含說明如何執行和使用範例的讀我檔案。 範例儲存機制的主要讀我檔案說明了如何參與。 

範例和範本是所有授權 MIT 授權。 請參閱根目錄中的 license.txt 檔。

電子郵件問題： sqlserversamples@microsoft.com。


## <a name="wideworldimporters-sample-database"></a>WideWorldImporters 範例資料庫

WideWorldImporters （生平） 是展示資料庫設計使用目前的功能、 工具和設計技巧的範例資料庫。 WideWorldImporters 下載包括 OLTP 與 OLAP 資料庫是內建至執行 SQL Server 2016 和更新版本。 

下載資料庫和範例位於[SQL Server 範例](https://github.com/Microsoft/sql-server-samples)GitHub 儲存機制。


- [WideWorldImporters 讀我檔案和範例](https://github.com/Microsoft/sql-server-samples/tree/master/samples/databases/wide-world-importers)GitHub 上。

- [教學課程： 載入 WideWorldImportersDW 到 Azure SQL 資料倉儲](/azure/sql-data-warehouse/load-data-wideworldimportersdw)

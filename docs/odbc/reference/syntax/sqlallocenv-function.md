**一致性**

 在 ODBC 3 中。*x*，ODBC 2。*x* 函數 **SQLAllocEnv** 已被 **SQLAllocHandle**取代。 如需詳細資訊，請參閱 [SQLAllocHandle 函數](../../../odbc/reference/syntax/sqlallochandle-function.md)。

# <a name="standard-data-stream-protocol"></a>標準資料流通訊協定
標準資料流通訊協定是一種方式在異質性的 Dbms 中存取資料。 事實上，標準資料流通訊協定已存在：

- ANSI/ISO 遠端資料庫存取 (RDA) 標準： [ISO/IEC 9579:2000](https://www.iso.org/iso/catalogue_detail.htm?csnumber=30615)。 雖然 ANSI/ISO 系統顯示承諾，但它尚未廣泛實作今天。

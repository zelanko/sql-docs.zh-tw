# <a name="what39s-new-in-odbc-38"></a>什麼&#39;s ODBC 3.8 的新功能
Windows 8 包含 ODBC 3.8 的更新的版本。 Windows 8 中的 ODBC 3.8 包括下列功能：  
  
-   [可感知驅動程式的連接共用](../../odbc/reference/develop-app/driver-aware-connection-pooling.md)  
  
-   [非同步執行 (通知方法)](../../odbc/reference/develop-app/asynchronous-execution-notification-method.md)  
  
-   [資料存取追蹤 (Windows 8)](https://msdn.microsoft.com/library/windows/desktop/hh829624.aspx)  
  
-   已加入 PowerShell 命令來協助您管理 ODBC 資料來源、 ODBC 驅動程式、 ODBC 的效能計數器，以及資料存取追蹤在命令列。  如需詳細資訊，請參閱[Windows 資料存取元件 PowerShell 命令](https://msdn.microsoft.com/library/windows/desktop/jj134064.aspx)。  
  
 Windows 7 包括 ODBC，ODBC 3.8 的更新的版本。 ODBC 3.8 包括下列功能：  
  
-   以非同步方式執行連接作業。 如需詳細資訊，請參閱[非同步執行 （輪詢方法）](../../odbc/reference/develop-app/asynchronous-execution-polling-method.md)。  
  
-   資料流的輸出參數。 如需詳細資訊，請參閱[使用 SQLGetData 擷取輸出參數](../../odbc/reference/develop-app/retrieving-output-parameters-using-sqlgetdata.md)。  
  
-   ODBC C 資料類型擴充性。 如需詳細資訊，請參閱[ODBC 中的 C 資料類型](../../odbc/reference/develop-app/c-data-types-in-odbc.md)。  
  
 驅動程式寫入器應該閱讀[3.8 驅動程式升級 3.5 驅動程式](../../odbc/reference/develop-driver/upgrading-a-3-5-driver-to-a-3-8-driver.md)。  
  
 非同步連接作業可供 ODBC 3.x 和 ODBC 2.x 應用程式使用 ODBC 3.8 驅動程式。  
  
 如需詳細資訊，請參閱[相容性比較表](../../odbc/reference/develop-app/compatibility-matrix.md)。

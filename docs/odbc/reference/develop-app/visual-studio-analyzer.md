# <a name="visual-studio-analyzer"></a>Visual Studio Analyzer
> [!IMPORTANT]  
>  Visual Studio Analyzer 的支援已移除開頭 （Visual Studio Analyzer 只包含在舊版的 Visual Studio）。 Windows 8 中。 如需疑難排解機制的替代方法，使用 BID 追蹤。  
  
 Microsoft® Visual Studio™ 分析器中提供的 ODBC 應用程式的高階檢視。 開發人員可以評估、 分析和偵錯結構、 效能及應用程式的觀點來看，而不是從元件或程式碼的檢視方塊的互動。 代表某種類型的應用程式的兩個元件之間互動的事件表單中，是由 Visual Studio 分析器所收集的資訊。 事件是由函式呼叫，並從函式，例如連線、 資料庫查詢和交易傳回所組成。

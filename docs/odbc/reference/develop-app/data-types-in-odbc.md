# <a name="data-types-in-odbc"></a>ODBC 中的資料類型
ODBC 會使用兩組資料類型：SQL 資料類型和 C 資料類型。 SQL 資料類型會在資料來源，並在應用程式中的 C 程式碼會使用 C 資料類型。

# <a name="cstring-class"></a>CString 類別
由於 Microsoft 中的**CString**類的物件® Visual C++®已簽名,並且 ODBC 函數中的字串參數沒有簽名,因此將**CString**物件傳遞給 ODBC 函數而不強制轉換它們的應用程式將收到編譯器警告。

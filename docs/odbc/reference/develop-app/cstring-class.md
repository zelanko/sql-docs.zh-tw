# <a name="cstring-class"></a>CString 類別
因為物件的**CString** Microsoft® C++® 中的類別已簽署，在 ODBC 函數的字串引數是不帶正負號，應用程式會通過**CString** ODBC 的功能，但不包含的物件將它們轉型將會收到編譯器警告。

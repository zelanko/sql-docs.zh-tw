# <a name="odbc-test"></a>ODBC 測試
Microsoft® ODBC 測試是 ODBC 的應用程式可供您測試 ODBC 驅動程式和 ODBC 驅動程式管理員。 ODBC 3.51 包含 ANSI 和 Unicode 相容的 ODBC 測試的版本。 對應的檔案如下所示：  
  
-   Odbcte32.exe 和 Gtrtst32.dll，ANSI 版本。  
  
-   Odbct32w.exe 和 Gtrts32w.dll，Unicode 版本。  
  
 若要使用 ODBC 測試，您必須了解 ODBC API 的 C 語言和 SQL。 如需有關 ODBC API 的詳細資訊，請參閱 < [ODBC 程式設計人員參考](../odbc/reference/odbc-programmer-s-reference.md)。  
  
 ODBC 測試計劃中現在包含先前隨附文件的本節中的 [說明] 主題。 開啟 Odbcte32.exe 或 Odbct32w.exe，開放**幫助**功能表，然後再按一下**說明主題**。  
  
 請注意，64 位元版本的這些應用程式，適用於 64 位元 Microsoft Windows 作業系統中，同名的 32 位元版本，即使它們是不同的檔案。 也就是 ODBC 測試 64 位元版本的 Unicode 版本的名稱是 odbct32w.exe。

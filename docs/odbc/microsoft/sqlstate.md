# <a name="sqlstate"></a>SQLSTATE
對於資料來源中發生的錯誤,ODBC 驅動程式將返回的本機錯誤映射到相應的 SQLSTATE。 對於驅動程式或驅動程式管理器檢測到的錯誤,ODBC 驅動程式或驅動程式管理器將生成相應的 SQLSTATE。

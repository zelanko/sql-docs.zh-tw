# <a name="sqlexecute-visual-foxpro-odbc-driver"></a>SQLExecute （Visual FoxPro ODBC 驅動程式）
> [!NOTE]  
>  本主題包含 Visual FoxPro ODBC 驅動程式特有的資訊。 如需此函式的一般資訊，請參閱底下的適當主題[ODBC 應用程式開發介面參考](../../odbc/reference/syntax/odbc-api-reference.md)。  
  
 支援： 完整  
  
 ODBC 應用程式開發介面相容性： 核心層級  
  
 執行已備妥的 SQL 陳述式 (已備妥的陳述式[SQLPrepare](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md))。 如果陳述式中存在的任何參數，驅動程式會使用目前的參數標記變數值。  
  
 如需詳細資訊，請參閱[SQLExecute](../../odbc/reference/syntax/sqlexecute-function.md)中*ODBC 程式設計人員參考*。

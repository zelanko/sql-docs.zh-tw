# <a name="deleting-a-visual-foxpro-data-source"></a>刪除 Visual FoxPro 資料來源
### <a name="to-delete-a-visual-foxpro-data-source"></a>若要刪除 Visual FoxPro 資料來源  
  
1.  在執行 Windows 2000 的電腦，開啟 Windows 控制台中，按兩下 [系統管理工具]。  
  
2.  按兩下資料來源 (ODBC) 若要開啟 ODBC 資料來源管理員 對話方塊。 Visual FoxPro ODBC 驅動程式或任何 ODBC 驅動程式軟體安裝後使用此圖示。  
  
    > [!NOTE]  
    >  如果您執行舊版 Windows，開啟 Windows 控制台中，按兩下 32 位元 ODBC 或 ODBC 開啟 ODBC 資料來源管理員 對話方塊。  
  
3.  在 [使用者 DSN] 或 [系統 DSN] 索引標籤中，選取您想要刪除的資料來源的名稱。  
  
4.  按一下 [移除]。  
  
5.  按一下 [確定] 以儲存變更並關閉 [ODBC 資料來源管理員] 對話方塊。

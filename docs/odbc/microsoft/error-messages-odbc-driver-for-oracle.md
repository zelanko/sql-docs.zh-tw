# <a name="error-messages-odbc-driver-for-oracle"></a>錯誤訊息 (ODBC Driver for Oracle)
> [!IMPORTANT]  
>  Windows 的未來版本將移除這項功能。 請避免在新的開發工作中使用這項功能，並規劃修改目前使用這項功能的應用程式。 相反地，使用所提供的 ODBC 驅動程式。  
  
 發生錯誤時，Microsoft ODBC Driver for Oracle 傳回 SQLSTATE （ODBC 錯誤碼） 和錯誤訊息。 從 驅動程式所偵測到的錯誤和從 Oracle 伺服器所傳回的錯誤，驅動程式會衍生這項資訊。  
  
 本節涵蓋下列主題：  
  
-   [ODBC Driver for Oracle 所傳回的訊息](../../odbc/microsoft/messages-returned-by-the-odbc-driver-for-oracle.md)

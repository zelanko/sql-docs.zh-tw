# <a name="sqlsetpos-visual-foxpro-odbc-driver"></a>SQLSetPos (Visual FoxPro ODBC Driver)
> [!NOTE]  
>  本主題包含 Visual FoxPro ODBC 驅動程式特有的資訊。 如需此函式的一般資訊，請參閱底下的適當主題[ODBC API 參考](../../odbc/reference/syntax/odbc-api-reference.md)。  
  
 支援： 完整  
  
 ODBC API 相容性： 層級 2  
  
 設定資料列集中的資料指標位置。 您可以使用**SQLSetPos**具有[SQLGetData](../../odbc/microsoft/sqlgetdata-visual-foxpro-odbc-driver.md)未繫結的資料行中擷取資料列之後的特定資料列遊標定位在資料列集中。  
  
 如需詳細資訊，請參閱 < [SQLSetPos](../../odbc/reference/syntax/sqlsetpos-function.md)中*ODBC 程式設計人員參考*。

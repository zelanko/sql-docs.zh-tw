# <a name="sqlfreeenv-visual-foxpro-odbc-driver"></a>SQLFreeEnv （Visual FoxPro ODBC 驅動程式）
> [!NOTE]  
>  本主題包含 Visual FoxPro ODBC 驅動程式特有的資訊。 如需此函式的一般資訊，請參閱底下的適當主題[ODBC 應用程式開發介面參考](../../odbc/reference/syntax/odbc-api-reference.md)。  
  
 支援： 完整  
  
 ODBC 應用程式開發介面相容性： 核心層級  
  
 關閉 Visual FoxPro ODBC 驅動程式，並釋放所有驅動程式相關聯的記憶體。  
  
 如需詳細資訊，請參閱[SQLFreeEnv](../../odbc/reference/syntax/sqlfreeenv-function.md)中*ODBC 程式設計人員參考*。

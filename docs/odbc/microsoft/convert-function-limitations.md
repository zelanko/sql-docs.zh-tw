# <a name="convert-function-limitations"></a>CONVERT 函式限制
類型轉換失敗會導致受影響的資料行設定為 Null。  
  
 日期或時間戳記資料類型都無法轉換成另一種資料類型 (或本身由 CONVERT 函數) 。

# <a name="visual-foxpro-terminology"></a>Visual FoxPro 術語
**資料庫**  
 在 Visual FoxPro，資料庫檔案的副檔名為.dbc，可以包含一或多個**資料表**。  
  
 **資料庫資料表**  
 在 Visual FoxPro，資料表，是與資料庫相關聯。 對比**免費資料表**。  
  
 **可用的資料表**  
 在 Visual FoxPro 時，不是與資料庫相關聯的資料表。  
  
 建立的.dbf 檔案在 FoxPro 版本 2.x 可用的資料表，除非是它會轉換成 Visual FoxPro 資料表，並加入至 Visual FoxPro 資料庫。 對比**資料庫資料表**。  
  
 **preparable 的 SQL 陳述式**  
 尚未處理的 SQL 陳述式**SQLPrepare**函式。 如需有關使用 Visual FoxPro ODBC 驅動程式的這個函式的詳細資訊，請參閱[SQLPrepare (Visual FoxPro ODBC Driver)](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md)。  
  
 **table**  
 在 Visual FoxPro，記錄會儲存在資料表中。 資料表的每個資料列都代表一筆記錄，以及資料表的資料行則代表記錄的欄位。 每個 Visual FoxPro 資料表會儲存在自己的檔案，副檔名為.dbf。 Visual FoxPro 資料表可以是與資料庫相關聯。  
  
 FoxPro 第 2 版。*x*資料表未與資料庫相關聯。

# <a name="visual-foxpro-terminology"></a>Visual FoxPro 術語
**資料庫**  
 Visual FoxPro，在資料庫檔案.dbc 副檔名，且可以包含一或多個**資料表**。  
  
 **資料庫資料表**  
 Visual FoxPro，在資料表，是與資料庫相關聯。 對比**可用資料表**。  
  
 **可用的資料表**  
 在 Visual FoxPro，不是與資料庫相關聯的資料表。  
  
 建立的.dbf 檔案在 FoxPro 版本 2.x 可用的資料表，除非是它會轉換成 Visual FoxPro 資料表並加入至 Visual FoxPro 資料庫。 對比**資料庫資料表**。  
  
 **可準備 SQL 陳述式**  
 尚未處理的 SQL 陳述式**SQLPrepare**函式。 如需此函式的 Visual FoxPro ODBC 驅動程式的詳細資訊，請參閱[SQLPrepare （Visual FoxPro ODBC 驅動程式）](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md)。  
  
 **table**  
 在 Visual FoxPro，記錄會儲存在資料表中。 資料表的每個資料列都代表記錄，以及資料表的資料行代表資料錄的欄位。 每個 Visual FoxPro 資料表儲存在它自己.dbf 副檔名的檔案。 Visual FoxPro 資料表可以是與資料庫相關聯。  
  
 FoxPro 第 2 版。*x*資料表不會與資料庫產生關聯。

# <a name="odbc-component-files"></a>ODBC 元件檔案
ODBC 桌面資料庫驅動程式 4.0 的每個元件所需的 ODBC 驅動程式檔案。  
  
 某些檔案需要多個元件。 如果您想要重新發佈任何 ODBC 桌面資料庫驅動程式與您的商業應用程式，必須重新發佈這些檔案。  
  
 這些檔案會安裝在 Windows 95 或更新版本，\Windows\System 目錄或 Windows 2000 與 Windows NT 3.51 或更新版本的 \Windows\System32 目錄。 如果 ODBC 檔案先前已安裝在不同的目錄，請確定您使用較新的 \Windows\System （或 System32） 中的檔案目錄。

# <a name="jet-40-uses-sql-92-reserved-words-list-when-extendedansisqlset"></a>Jet 4.0 使用 sql-92 保留的字清單時 ExtendedAnsiSQL_Set
當開啟 ExtendedAnsiSQL 旗標時，Jet 4.0 會使用 sql-92 保留的字清單。 嘗試使用 SQL 92 保留字，不具引號的物件名稱會造成語法錯誤。 ExtendedAnsiSQL 旗標為關閉時，新的保留的字可用來當作與之前的物件名稱。

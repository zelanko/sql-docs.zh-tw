# <a name="drop-table-statement-limitations"></a>卸除資料表陳述式的限制
使用 Microsoft Excel 5.0、 7.0、 或 97 驅動程式時，DROP TABLE 陳述式會清除工作表，但不會刪除工作表的名稱。 因為工作表名稱仍存在於活頁簿中，另一個工作表無法建立具有相同名稱。

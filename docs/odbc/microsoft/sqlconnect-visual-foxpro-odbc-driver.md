# <a name="sqlconnect-visual-foxpro-odbc-driver"></a>SQLConnect (Visual FoxPro ODBC Driver)
> [!NOTE]  
>  本主題包含 Visual FoxPro ODBC 驅動程式特有的資訊。 如需此函式的一般資訊，請參閱底下的適當主題[ODBC API 參考](../../odbc/reference/syntax/odbc-api-reference.md)。  
  
 支援：完整  
  
 ODBC API 相容性：核心層級  
  
 連接到資料來源，這可以是[資料庫](../../odbc/microsoft/visual-foxpro-terminology.md)或 目錄[資料表](../../odbc/microsoft/visual-foxpro-terminology.md)。 Visual FoxPro ODBC 驅動程式會忽略*szUID*， *cbUID*， *szAuthStr*，以及*cbAuthStr*引數。  
  
 如需詳細資訊，請參閱 < [SQLConnect](../../odbc/reference/syntax/sqlconnect-function.md)中*ODBC 程式設計人員參考*。

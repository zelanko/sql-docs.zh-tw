# <a name="visual-foxpro-odbc-driver"></a>Visual FoxPro ODBC 驅動程式
Microsoft Visual FoxPro 是功能強大物件導向環境，來建構和應用程式開發的資料庫。 Microsoft Visual FoxPro ODBC 驅動程式可讓應用程式來開啟查詢和更新 Visual FoxPro 及較早版本 FoxPro 透過開啟資料庫連接 (ODBC) 介面中的資料。  
  
 比方說的 Microsoft Visual FoxPro ODBC 驅動程式，您可以執行下列動作：  
  
-   使用 Microsoft 查詢來查詢及更新 Visual FoxPro 資料從 Microsoft Excel 工作表。  
  
-   建立使用 Microsoft Word Visual FoxPro 資料合併字母。  
  
-   查詢並更新 Visual FoxPro 檢視和資料表從 Microsoft Access。  
  
-   使用 Visual FoxPro 為 Microsoft Visual Basic、 Microsoft Visual c + + 和 C 應用程式的資料存放區。

# <a name="visual-foxpro-odbc-driver"></a>Visual FoxPro ODBC Driver
Microsoft Visual FoxPro 是強大的物件導向環境，適用于資料庫結構和應用程式開發。 Microsoft Visual FoxPro ODBC 驅動程式可讓應用程式透過開放式資料庫連接（ODBC）介面，開啟、查詢和更新 Visual FoxPro 和舊版 FoxPro 中的資料。  
  
 例如，使用 Microsoft Visual FoxPro ODBC 驅動程式，您可以執行下列動作：  
  
-   使用 Microsoft Query 來查詢和更新 Microsoft Excel 工作表中的 Visual FoxPro 資料。  
  
-   使用 Visual FoxPro 資料搭配 Microsoft Word 建立合併列印字母。  
  
-   從 Microsoft Access 查詢和更新 Visual FoxPro views 和資料表。  
  
-   使用 Visual FoxPro 做為 Microsoft Visual Basic、Microsoft Visual C++ 和 C 應用程式的資料存放區。

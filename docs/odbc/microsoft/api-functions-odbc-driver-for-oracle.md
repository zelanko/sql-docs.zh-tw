# <a name="api-functions-odbc-driver-for-oracle"></a>API 函式 (ODBC Driver for Oracle)
> [!IMPORTANT]  
>  Windows 的未來版本將移除這項功能。 請避免在新的開發工作中使用這項功能，並規劃修改目前使用這項功能的應用程式。 相反地，使用所提供的 ODBC 驅動程式。  
  
 Microsoft ODBC Driver for Oracle 支援的核心層級，層級 1 和層級 2 API 函式。 這些函式會列在[ODBC 一致性層級](../../odbc/microsoft/odbc-driver-for-oracle-conformance-levels.md)。  
  
-   [核心層級介面](../../odbc/microsoft/core-level-api-functions-odbc-driver-for-oracle.md)(CLI) 一致性提供 ISO CLI 規格中定義的功能，以及開啟群組 CLI 規格中定義的必要功能。  
  
-   [層級 1](../../odbc/microsoft/level-1-api-functions-odbc-driver-for-oracle.md)一致性提供核心層級介面的功能，以及其他功能，例如交易。  
  
-   [層級 2](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md)一致性提供層級 1 的功能，以及其他功能，例如書籤、 動態參數，以及非同步執行的 ODBC 函式。

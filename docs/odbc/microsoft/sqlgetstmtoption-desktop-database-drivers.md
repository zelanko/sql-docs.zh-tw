# <a name="sqlgetstmtoption-desktop-database-drivers"></a>SQLGetStmtOption (桌面資料庫驅動程式)
SQL_GETBOOKMARK 的 *fOption* 所傳回的書簽只有在查詢開啟時才有效，而且會在重新發出查詢時失效。 不支援持續性書簽。

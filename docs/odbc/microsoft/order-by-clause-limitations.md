# <a name="order-by-clause-limitations"></a>ORDER BY 子句限制
如果 SELECT 語句包含 GROUP BY 子句和 ORDER BY 子句，ORDER BY 子句只能包含結果集中的資料行或 GROUP BY 子句中的運算式。

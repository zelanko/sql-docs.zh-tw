# <a name="creating-and-dropping-views-and-procedures-using-extendedansisql"></a>使用 ExtendedAnsiSQL 來建立及置放檢視和程序
ExtendedAnsiSQL 旗標為開啟時，Jet 4.0 支援建立和卸除檢視和程序透過 SQL 的能力。 這些檢視和程序將不會顯示透過存取的。

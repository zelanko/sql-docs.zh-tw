# <a name="using-synonyms-with-stored-procedures"></a>搭配使用同義字與預存程序
> [!IMPORTANT]  
>  Windows 的未來版本將移除這項功能。 請避免在新的開發工作中使用這項功能，並規劃修改目前使用這項功能的應用程式。 相反地，使用所提供的 ODBC 驅動程式。  
  
 Microsoft ODBC Driver for Oracle 版本 2.0 和 2.5 不支援同義字呼叫 Oracle 預存程序。 如預期般與其他的 Oracle 資料庫物件，例如資料表搭配使用時，就會運作的同義字。

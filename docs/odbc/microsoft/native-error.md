# <a name="native-error"></a>原生錯誤
正本機錯誤代碼由桌面資料庫驅動程式生成。 負錯誤代碼由微軟噴氣機生成。 有關詳細資訊,請參閱 Microsoft*噴氣資料庫引擎程式師指南*。

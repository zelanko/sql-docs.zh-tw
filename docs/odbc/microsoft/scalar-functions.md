# <a name="scalar-functions"></a>純量函數
Microsoft ODBC 桌面資料庫驅動程式支援以下標量函數:  
  
||||  
|-|-|-|  
|CONCAT|LCASE|RIGHT|  
|CONVERT|LEFT|RTRIM|  
|CURDATE|LENGTH|SUBSTRING|  
|CURTIME|LOCATE|UCASE|  
|DATABASE|LTRIM|USER|  
|DAYOFMONTH|MOD|年|  
|DAYOFWEEK|月||  
  
 有關標量函數的參數和返回值的資訊,請參閱*ODBC 程式師參考*的附錄 F。

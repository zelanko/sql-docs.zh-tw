# <a name="scalar-functions"></a>純量函數
Microsoft ODBC 桌面資料庫驅動程式支援下列純量函數：  

:::row:::
    :::column:::
        CONCAT  
        CONVERT  
        CURDATE  
        CURTIME  
        DATABASE  
        DAYOFMONTH  
        DAYOFWEEK  
    :::column-end:::
    :::column:::
        LCASE  
        LEFT  
        LENGTH  
        LOCATE  
        LTRIM  
        MOD  
        月  
    :::column-end:::
    :::column:::
        RIGHT  
        RTRIM  
        SUBSTRING  
        UCASE  
        USER  
        年  
    :::column-end:::
:::row-end:::

 如需純量函數之引數和傳回值的相關資訊，請參閱*ODBC 程式設計人員參考*的附錄 F。

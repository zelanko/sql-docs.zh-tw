  本節說明如何在 Visual c [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] # .net 中設計管理物件[!INCLUDE[msCoName](../../includes/msconame-md.md)] （SMO）的程式。
  
|主題|描述|  
|-----------|-----------------|

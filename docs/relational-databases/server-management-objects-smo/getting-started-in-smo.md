# <a name="getting-started-in-smo"></a>SMO 使用者入門
[!INCLUDE[appliesto-ss-asdb-asdw-xxx-md](../../includes/appliesto-ss-asdb-asdw-xxx-md.md)]

此頁面會提供連結，主題旨在協助您開始使用 SQL Server 管理物件 (SMO)。 這些主題會列出封面 SMO 物件階層中，撰寫 SMO、 SMO 及不同的程式語言特定的程式設計工作的程式。  
 
##<a name="installing-smo"></a>安裝 SMO
[安裝 SMO](installing-smo.md)提供指示如何安裝和參考 SMO 組件。

## <a name="object-model"></a>物件模型  
[物件模型](../../relational-databases/server-management-objects-smo/smo-object-model.md)描述 SMO 物件階層以及物件與關聯另一個。  
  
## <a name="programming-languages"></a>程式語言  
[程式設計語言](../../relational-databases/server-management-objects-smo/smo-programming-languages.md)描述程式設計環境，並包含開始使用 C# 中撰寫 SMO 程式的詳細程序。  
  
## <a name="general-programming-in-smo"></a>SMO 中的一般程式設計  
[SMO 中的一般程式設計](../../relational-databases/server-management-objects-smo/create-program/creating-smo-programs.md)介紹使用 SMO 進行程式設計。 本主題描述如何連接到 SQL Server 執行個體以及如何使用屬性、 方法和集合。 更進階的主題會描述資料類型、交易、設定擷取模式和事件以及例外狀況處理。  
  
## <a name="programming-specific-tasks"></a>程式設計特有的工作  
[特定工作的程式設計](../../relational-databases/server-management-objects-smo/tasks/programming-specific-tasks.md)包括概念和程序的相關特定工作使用 SMO 進行程式設計的方式。 此主題也描述 SQL Server 的完整程式設計的管理。

# <a name="servernetworkprotocol-class"></a>ServerNetworkProtocol 類別
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  [ServerNetworkProtocol 類別](../../../relational-databases/wmi-provider-configuration-classes/servernetworkprotocol-class/servernetworkprotocol-class.md)代表網路安裝的執行個體上的通訊協定[!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] ，透過網路啟用與用戶端通訊。  
  
 具有[ServerNetworkProtocol 類別](../../../relational-databases/wmi-provider-configuration-classes/servernetworkprotocol-class/servernetworkprotocol-class.md)，您可以：  
  
-   檢查參考之伺服器網路通訊協定的名稱、版本、.dll 檔案、屬性數目和顯示名稱。  
  
-   參考網路通訊協定的屬性和 IP 位址。  
  
-   啟用和停用伺服器網路通訊協定。

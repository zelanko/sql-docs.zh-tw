# <a name="serversettings-class"></a>ServerSettings 類別
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/applies-to-version/sqlserver.md)]
  [ServerSettings 類別](../../../relational-databases/wmi-provider-configuration-classes/serversettings-class/serversettings-class.md)代表實例上的伺服器設定 [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 。  
  
 [ServerSettings 類別](../../../relational-databases/wmi-provider-configuration-classes/serversettings-class/serversettings-class.md)可讓您執行下列工作：  
  
-   存取 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 執行個體的一般旗標設定。  
  
-   設定 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 執行個體的預設值。  
  
-   設定 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 執行個體的安全性憑證。

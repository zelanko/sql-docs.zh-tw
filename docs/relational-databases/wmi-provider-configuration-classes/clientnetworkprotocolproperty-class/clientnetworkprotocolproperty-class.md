# <a name="clientnetworkprotocolproperty-class"></a>ClientNetworkProtocolProperty 類別
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)][ClientNetworkProtocolProperty 類別](../../../relational-databases/wmi-provider-configuration-classes/clientnetworkprotocolproperty-class/clientnetworkprotocolproperty-class.md)代表用戶端網路通訊協定的屬性。  
  
 與[ClientNetworkProtocolProperty 類別](../../../relational-databases/wmi-provider-configuration-classes/clientnetworkprotocolproperty-class/clientnetworkprotocolproperty-class.md)，您可以執行下列工作：  
  
-   設定索引值，以便從屬性清單中選取用戶端網路通訊協定屬性。  
  
-   檢查目前屬性所屬的通訊協定。  
  
-   檢查目前屬性的名稱、類型和值。  
  
-   修改目前屬性的屬性字串值和數值。  
  
-   設定目前屬性的旗標。

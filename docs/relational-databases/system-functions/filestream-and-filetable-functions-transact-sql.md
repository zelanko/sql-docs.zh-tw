# <a name="filestream-and-filetable-functions-transact-sql"></a>Filestream 和 FileTable 函數 (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  本節描述與 FILESTREAM 和 FileTable 功能相關的系統函數。  
  
 [FileTableRootPath &#40;Transact-SQL&#41;](../../relational-databases/system-functions/filetablerootpath-transact-sql.md)  
 傳回特定 FileTable 或目前資料庫的根層級 UNC 路徑。  
  
 [GetFileNamespacePath &#40;Transact-SQL&#41;](../../relational-databases/system-functions/getfilenamespacepath-transact-sql.md)  
 傳回 FileTable 中檔案或目錄的 UNC 路徑。  
  
 [GetPathLocator &#40;Transact-SQL&#41;](../../relational-databases/system-functions/getpathlocator-transact-sql.md)  
 傳回 FileTable 中指定之檔案或目錄的路徑定位程式識別碼值。  
  
 [PathName &#40;Transact-SQL&#41;](../../relational-databases/system-functions/pathname-transact-sql.md)  
 傳回 FILESTREAM 二進位大型物件 (BLOB) 的路徑。  
  
 如需有關 FileTable 的詳細資訊，請參閱 [FileTables &#40;SQL Server&#41;](../../relational-databases/blob/filetables-sql-server.md)。

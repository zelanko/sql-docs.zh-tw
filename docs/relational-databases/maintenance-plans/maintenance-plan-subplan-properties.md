# 維護計畫 (子計畫屬性)
  使用 [維護計畫 (子計畫屬性)] 對話方塊可指定維護計畫子計畫的名稱、描述和排程。 按一下維護計畫 [設計] 索引標籤上的 [子計畫屬性]，或按兩下設計工具方格中的子計畫，來顯示這個對話方塊。  
  
## 選項  
 **名稱**  
 輸入子計畫的名稱。 名稱限制為 128 個字元。  
  
 **描述**  
 輸入子計畫的描述。 描述限制為 512 個字元。  
  
 **排程**  
 按一下排程圖示指定何時執行子計畫工作，或按一下有紅色 x 的排程圖示，移除現有子計畫排程。

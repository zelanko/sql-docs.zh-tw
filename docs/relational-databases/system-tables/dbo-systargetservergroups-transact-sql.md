# <a name="dbosystargetservergroups-transact-sql"></a>dbo.systargetservergroups (Transact-SQL)

  記錄在這個多伺服器環境中，目前編列了哪些目標伺服器群組。  
  
  
|資料行名稱|資料類型|描述|  
|-----------------|---------------|-----------------|  
|**servergroup_id**|**int**|伺服器群組識別碼|  
|**name**|**sysname**|伺服器群組名稱|

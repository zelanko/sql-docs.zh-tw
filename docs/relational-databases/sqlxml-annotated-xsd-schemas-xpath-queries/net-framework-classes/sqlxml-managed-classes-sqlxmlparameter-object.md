# <a name="sqlxml-managed-classes---sqlxmlparameter-object"></a>SQLXML Managed 類別-SqlXmlParameter 物件
[!INCLUDE[appliesto-ss-asdb-xxxx-xxx-md](../../../includes/appliesto-ss-asdb-xxxx-xxx-md.md)]
SqlXmlParameter 物件支援這些屬性：  
  
 名稱  
 參數的名稱。 可以將參數傳遞給命令。 呼叫 CreateParameter 方法 SqlXmlCommand 物件的建立參數物件。  
  
 值  
 參數的值。 如需實用範例，請參閱[執行 SQL 查詢 &#40;SQLXML Managed 類別 &#41;](../../../relational-databases/sqlxml-annotated-xsd-schemas-xpath-queries/net-framework-classes/executing-sql-queries-sqlxml-managed-classes.md).

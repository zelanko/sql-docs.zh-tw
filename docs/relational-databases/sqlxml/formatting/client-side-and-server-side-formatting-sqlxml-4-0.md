# <a name="client-side-and-server-side-formatting-sqlxml-40"></a>用戶端和伺服器端格式 (SQLXML 4.0)
[!INCLUDE[appliesto-ss-asdb-xxxx-xxx-md](../../../includes/appliesto-ss-asdb-xxxx-xxx-md.md)]
  SQLXML 提供兩個方式來格式化 XML 文件：  
  
-   伺服器端格式  
  
-   用戶端格式  
  
 本節包含在用戶端或伺服器端，從針對資料庫執行之查詢所產生之資料[!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]列集的 XML 檔案格式設定資訊。

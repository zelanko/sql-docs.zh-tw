# <a name="annotated-xdr-schemas-deprecated-in-sqlxml-40"></a>註解式 XDR 結構描述 (在 SQLXML 4.0 中已被取代)
[!INCLUDE[appliesto-ss-asdb-xxxx-xxx-md](../../../includes/appliesto-ss-asdb-xxxx-xxx-md.md)]
  註解式 XDR 結構描述已被取代。 這項功能未來將不會繼續開發， 請避免在新的開發工作中使用此功能。 請改用註解式 XSD 結構描述。 您可以將現有註解式 XDR 結構描述轉換為 XSD 結構描述。 如需詳細資訊，請參閱[將批註式 XDR 架構轉換為對等的 XSD 架構 &#40;SQLXML 4.0&#41;](../../../relational-databases/sqlxml/annotated-xsd-schemas/converting-annotated-xdr-schemas-to-equivalent-xsd-schemas-sqlxml-4-0.md)。 如需已被取代之 XDR 架構的詳細資訊，請參閱[SQL Server 2008 檔](https://go.microsoft.com/fwlink/?LinkId=202011)。

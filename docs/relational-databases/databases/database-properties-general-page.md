# <a name="database-properties-general-page"></a>資料庫屬性 (一般頁面)
 [!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]
  使用此頁面來檢視或修改選取之資料庫的屬性。  
  
## <a name="options"></a>選項。  
 **上次資料庫備份**  
 顯示上一次備份資料庫的日期。  
  
 **上次資料庫記錄備份**  
 顯示上一次備份資料庫交易記錄的日期。  
  
 **名稱**  
 顯示資料庫的名稱。  
  
 **狀態**  
 顯示資料庫狀態。 如需詳細資訊，請參閱 [Database States](../../relational-databases/databases/database-states.md)。  
  
 **擁有者**  
 顯示資料庫擁有者的名稱。 可以在 **[檔案]** 頁面上變更擁有者。  
  
 **建立日期**  
 顯示建立資料庫的日期和時間。  
  
 **大小**  
 顯示資料庫的大小 (以 MB 計)。  
  
 **可用空間**  
 顯示資料庫中可用的空間量 (以 MB 表示)。  
  
 **使用者數目**  
 顯示為資料庫所設定的使用者數目。  
  
 **定序名稱**  
 顯示資料庫所使用的定序。 定序可以在 **[選項]** 頁面上變更。

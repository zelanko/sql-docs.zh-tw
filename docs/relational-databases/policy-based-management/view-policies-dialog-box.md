# <a name="view-policies-dialog-box"></a>檢視原則對話方塊
  使用此對話方塊可檢視對這個目標有效的原則。 此對話方塊會顯示相關的原則、其原則類別目錄以及有關原則的資訊。  
  
## <a name="options"></a>選項  
 **原則健全狀態**  
 指出與這個目標有關之原則的狀態。 當狀態未知時會保留空白。  
  
 **有效**  
 指出此原則目前對目標是否有效。  
  
 **原則**  
 每一個適用的原則名稱。  
  
 **類別目錄**  
 選定原則所屬的原則類別目錄。  
  
 **評估模式**  
 具名原則的評估模式。  
  
 **記錄**  
 按一下此超連結可顯示執行記錄報表。  
  
 **評估**  
 按一下此超連結可開啟 [評估原則] 對話方塊來執行此原則。  
  
 **上次執行**  
 上次執行此原則的日期和時間。

# <a name="results-detailed-view-dialog-box"></a>結果詳細檢視對話方塊
  當您使用 **[評估原則]** 對話方塊來執行原則並按一下 **[評估]**之後，此對話方塊會顯示原則評估的結果。 此對話方塊是唯讀的，而且可幫助您了解哪一個部分的屬性運算式可能失敗。  
  
## <a name="options"></a>選項  
 **AndOr**  
 當有一個以上的屬性運算式存在時，指出屬性運算式為累積或替代式。  
  
 **結果**  
 指出屬性運算式為成功或失敗的圖示。  
  
 **欄位**  
 正在模型化之 Facet 的屬性。  
  
 **運算子**  
 運算式的運算子，例如 **=** 或 **Like**。  
  
 **預期的值**  
 將使得屬性運算式成功之欄位的值。  
  
 **實際值**  
 此原則所偵測之欄位的值。  
  
 **原則描述**  
 此原則的描述。  
  
 **其他說明**  
 按一下此超連結可開啟與此原則相關的網頁。 當建立此原則，而此原則可能為空白或無法使用時，會設定 [其他說明] 超連結。

# <a name="setnclob-method-sqlserverpreparedstatement"></a>setNClob 方法 (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  將指定的參數設定為指定的物件。  
  
## <a name="overload-list"></a>多載清單  
  
|名稱|Description|  
|----------|-----------------|  
|[setNClob 方法&#40;int，java.sql.NClob&#41;](../../../connect/jdbc/reference/setnclob-method-int-java-sql-nclob.md)|將指定的參數設定為指定的 java.sql.NClob 物件。|  
|[setNClob 方法&#40;int，java.io.Reader&#41;](../../../connect/jdbc/reference/setnclob-method-int-java-io-reader.md)|將指定的參數設定為指定的 java.io.Reader 物件。|  
|[setNClob 方法&#40;int，java.io.Reader，long&#41;](../../../connect/jdbc/reference/setnclob-method-int-java-io-reader-long.md)|指定的參數設定為指定的 java.io.Reader 物件，也就是指定的字元數。|

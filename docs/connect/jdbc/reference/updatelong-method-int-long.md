# <a name="updatelong-method-int-long"></a>updateLong 方法 (int, long)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  更新指定的資料行與**長**給定的資料行索引的值。  
  
## <a name="syntax"></a>語法  
  
```  
  
public void updateLong(int index,  
                       long x)  
```  
  
#### <a name="parameters"></a>參數  
 *索引*  
  
 **Int** ，指出資料行索引。  
  
 *x*  
  
 A**長**值。

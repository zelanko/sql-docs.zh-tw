# <a name="getblob-method-sqlservercallablestatement"></a>getBlob 方法 (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  擷取指定之 JDBC BLOB 參數的值來當做 Blob 物件使用 Java 程式語言。  
  
## <a name="overload-list"></a>多載清單  
  
|名稱|Description|  
|----------|-----------------|  
|[getBlob (int)](../../../connect/jdbc/reference/getblob-method-int.md)|擷取指定之 JDBC BLOB 參數的值來當做 Java 程式語言，使用給定的參數索引 Blob 物件。|  
|[getBlob (java.lang.String)](../../../connect/jdbc/reference/getblob-method-java-lang-string.md)|當做 Blob 物件中給定的參數名稱使用 Java 程式語言，擷取指定之 JDBC BLOB 參數的值。|

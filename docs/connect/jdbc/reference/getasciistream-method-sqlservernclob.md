# <a name="getasciistream-method-sqlservernclob"></a>getAsciiStream 方法 (SQLServerNClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  擷取**NCLOB**這由所指定值**NClob**當做 ASCII 資料流的物件。  
  
## <a name="syntax"></a>語法  
  
```  
  
public java.sql.InputStream getAsciiStream()  
```  
  
## <a name="return-value"></a>傳回值  
 包含 NCLOB 資料 InputStream 物件。  
  
## <a name="exceptions"></a>例外狀況  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>備註  
 GetAsciiStream 方法 java.sql.SQLServerNClob 介面中所指定此 getAsciiStream 方法。

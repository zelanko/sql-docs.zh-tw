# <a name="getdrivername-method-sqlserverdatabasemetadata"></a>getDriverName 方法 (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  擷取這個 JDBC Driver 的名稱。  
  
## <a name="syntax"></a>語法  
  
```  
  
public java.lang.String getDriverName()  
```  
  
## <a name="return-value"></a>傳回值  
 **String**，包含 JDBC Driver名稱。  
  
## <a name="exceptions"></a>例外狀況  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>備註  
 這個 getDriverName 方法是由 java.sql.DatabaseMetaData 介面中的 getDriverName 方法指定。

# <a name="getlabel-method-sqlserversavepoint"></a>getLabel 方法 (SQLServerSavepoint)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  取得儲存點標籤的名稱。  
  
## <a name="syntax"></a>語法  
  
```  
  
public java.lang.String getLabel()  
```  
  
## <a name="return-value"></a>傳回值  
 **String**，包含此儲存點標籤的名稱。

# <a name="type_ss_direct_forward_only-field-sqlserverresultset"></a>TYPE_SS_DIRECT_FORWARD_ONLY 欄位 (SQLServerResultSet)

## <a name="syntax"></a>語法  
  
```  
  
public static final int TYPE_SS_DIRECT_FORWARD_ONLY  
```  
  
## <a name="field-value"></a>欄位值  
 為 2003 的 **int** 值。

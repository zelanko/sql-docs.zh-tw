# <a name="getmajorversion-method-sqlserverdriver"></a>getMajorVersion 方法 (SQLServerDriver)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  傳回 [!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)] 的主要版本號碼。  
  
## <a name="syntax"></a>語法  
  
```  
  
public int getMajorVersion()  
```  
  
## <a name="return-value"></a>傳回值  
 **int**，包含 JDBC 驅動程式的主要版本號碼。  
  
## <a name="remarks"></a>備註  
 這個 getMajorVersion 方法是由 java.sql.Driver 介面中的 getMajorVersion 方法所指定。

# <a name="gettimestamp-method-sqlservercallablestatement"></a>getTimestamp 方法 (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  使用 Java 程式語言，擷取指定之參數的值來當做 java.sql.Timestamp 物件。  
  
## <a name="overload-list"></a>多載清單  
  
|名稱|描述|  
|----------|-----------------|  
|[getTimestamp (int)](../../../connect/jdbc/reference/gettimestamp-method-int.md)|在 Java 程式語言中使用給定的參數索引，擷取指定之參數的值來當做 java.sql.Timestamp 物件。|  
|[getTimestamp (int, java.util.Calendar)](../../../connect/jdbc/reference/gettimestamp-method-int-java-util-calendar.md)|使用 Java 程式設計語言，並配合所指定參數索引透過 Calendar 物件來擷取指定參數的值作為 java.sql.Timestamp 物件。|  
|[getTimestamp (java.lang.String)](../../../connect/jdbc/reference/gettimestamp-method-java-lang-string.md)|在 Java 程式語言中使用給定的參數名稱，擷取指定之參數的值來當做 java.sql.Timestamp 物件。|  
|[getTimestamp (java.lang.String, java.util.Calendar)](../../../connect/jdbc/reference/gettimestamp-method-java-lang-string-java-util-calendar.md)|使用 Java 程式設計語言，並配合所指定參數名稱透過 Calendar 物件來擷取指定參數的值作為 java.sql.Timestamp 物件。|

# <a name="type_ss_scroll_dynamic-field-sqlserverresultset"></a>TYPE_SS_SCROLL_DYNAMIC 欄位 (SQLServerResultSet)

## <a name="syntax"></a>語法  
  
```  
  
public static final int TYPE_SS_SCROLL_DYNAMIC
```  

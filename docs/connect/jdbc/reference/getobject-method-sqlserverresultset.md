# <a name="getobject-method-sqlserverresultset"></a>getObject 方法 (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

|----------|-----------------|  
|[getObject (int)](../../../connect/jdbc/reference/getobject-method-int-sqlserverresultset.md)|取得值，這個目前的資料列內指定之資料行索引的[SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md)物件為 Java 程式語言中的物件。|  
|[getObject （int，java.util.Map）](../../../connect/jdbc/reference/getobject-method-int-java-util-map-sqlserverresultset.md)|取得值，這個目前的資料列內指定之資料行索引的[SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md)物件為 Java 程式語言，並透過給定的 Map 物件中的物件。|  
|[getObject (java.lang.String)](../../../connect/jdbc/reference/getobject-method-java-lang-string-sqlserverresultset.md)|取得值，這個目前的資料列內指定之資料行名稱的[SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md)物件為 Java 程式語言中的物件。|

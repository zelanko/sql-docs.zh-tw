# <a name="setmultisubnetfailover-method-sqlserverdatasource"></a>setMultiSubnetFailover 方法 (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  設定 **multiSubnetFailover** 連接屬性的值。  
  
## <a name="syntax"></a>語法  
  
```vb  
public void setMultiSubnetFailover(boolean multiSubnetFailover);  
```  
  
#### <a name="parameters"></a>參數  
 *multiSubnetFailover*  
  
 **multiSubnetFailover** 連接屬性的新值。  
  
## <a name="remarks"></a>備註  
 如需 **multiSubnetFailover** 連接屬性的詳細資訊，請參閱[設定連接屬性](../../../connect/jdbc/setting-the-connection-properties.md)。

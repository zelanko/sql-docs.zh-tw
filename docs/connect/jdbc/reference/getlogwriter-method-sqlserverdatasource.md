# <a name="getlogwriter-method-sqlserverdatasource"></a>getLogWriter 方法 (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  此方法僅供內部使用。 如需記錄的詳細資訊，請參閱[追蹤驅動程式作業](../../../connect/jdbc/tracing-driver-operation.md)。  
  
## <a name="syntax"></a>語法  
  
```  
  
public java.io.PrintWriter getLogWriter()  
```  
  
## <a name="return-value"></a>傳回值  
 PrintWriter 物件。  
  
## <a name="remarks"></a>備註  
 這個 getLogWriter 方法是由 javax.sql.DataSource 介面中 getLogWriter 方法所指定的。

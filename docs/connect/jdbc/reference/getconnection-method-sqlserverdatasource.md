|[屬性]|Description|  
|----------|-----------------|  
|[getConnection ()](../../../connect/jdbc/reference/getconnection-method.md)|嘗試與這個 [SQLServerDataSource](../../../connect/jdbc/reference/sqlserverdatasource-class.md) 物件所代表的資料來源建立連線。|  
|[getConnection (java.lang.String, java.lang.String)](../../../connect/jdbc/reference/getconnection-method-java-lang-string-java-lang-string.md)|嘗試與這個 [SQLServerDataSource](../../../connect/jdbc/reference/sqlserverdatasource-class.md) 物件所代表的資料來源建立連接，其方式是使用指定的使用者名稱和密碼。|

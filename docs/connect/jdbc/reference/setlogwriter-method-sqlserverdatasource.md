# <a name="setlogwriter-method-sqlserverdatasource"></a>setLogWriter 方法 (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  此方法僅供內部使用。 如需記錄的詳細資訊，請參閱[追蹤驅動程式作業](../../../connect/jdbc/tracing-driver-operation.md)。  
  
## <a name="syntax"></a>語法  
  
```  
  
public void setLogWriter(java.io.PrintWriter out)  
```  
  
#### <a name="parameters"></a>參數  
 *Out*  
  
 PrintWriter 物件。  
  
## <a name="remarks"></a>備註  
 這個 setLogWriter 方法是由 javax.sql.DataSource 介面中 setLogWriter 方法所指定的。

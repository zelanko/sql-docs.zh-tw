# <a name="setclob-method-sqlserverpreparedstatement"></a>setClob 方法 (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  將指定的參數設定為 java.sql.Clob 物件。  
  
## <a name="overload-list"></a>多載清單  
  
|名稱|Description|  
|----------|-----------------|  
|[setClob 方法 &#40; int，java.io.Reader &#41;](../../../connect/jdbc/reference/setclob-method-int-java-io-reader.md)|將指定的參數設定為 java.sql.Clob 物件。|  
|[setClob 方法 &#40; int，java.sql.Clob &#41;](../../../connect/jdbc/reference/setclob-method-int-java-sql-clob.md)|將指定的參數設定為給定的 java.sql.Clob 物件。|  
|[setClob 方法 &#40; int，java.io.Reader，long &#41;](../../../connect/jdbc/reference/setclob-method-int-java-io-reader-long.md)|設定指定的參數給 java.sql.Clob 的物件，也就是指定的字元數。|  
  
## <a name="remarks"></a>備註  
 無伺服器驅動程式行為。 如需詳細資訊，請參閱 JSE 6 API 規格。

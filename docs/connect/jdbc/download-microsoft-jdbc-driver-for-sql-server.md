# <a name="download-microsoft-jdbc-driver-for-sql-server"></a>下載 Microsoft JDBC Driver for SQL Server

![下載 DownArrow Circled](../../ssdt/media/download.png)[若要下載 JDBC 驅動程式](../sql-connection-libraries.md#anchor-20-drivers-relational-access)
 
## <a name="using-the-jdbc-driver-with-maven-central"></a>使用 JDBC 驅動程式與 Maven 中央
JDBC 驅動程式可以藉由新增為下列程式碼 POM.xml 檔案中的相依性加入至 Maven 專案：

```xml
<dependency>
    <groupId>com.microsoft.sqlserver</groupId>
    <artifactId>mssql-jdbc</artifactId>
    <version>6.2.1.jre8</version>
</dependency>
```

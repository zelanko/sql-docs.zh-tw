# <a name="sql-server-and-adonet"></a>SQL Server 和 ADO.NET

[!INCLUDE[Driver_ADONET_Download](../../../includes/driver_adonet_download.md)]

## <a name="microsoftdatasqlclient"></a>Microsoft.Data.SqlClient

<xref:Microsoft.Data.SqlClient> 讓您能夠存取 SQL Server 版本，其中封裝了資料庫專屬的通訊協定。 <xref:Microsoft.Data.SqlClient> 包含表格式資料流 (TDS) 剖析器，可以直接與 SQL Server 通訊。  
  
> [!NOTE]
> 若要使用適用於 SQL Server 的 Microsoft SqlClient Data Provider，應用程式必須參考 <xref:Microsoft.Data.SqlClient> 命名空間。

# <a name="xml-data-in-sql-server"></a>SQL Server 中的 XML 資料

[!INCLUDE[Driver_ADONET_Download](../../../includes/driver_adonet_download.md)]

SQL Server 會在 .NET 內部公開 SQLXML 的功能。 開發人員可以撰寫從 SQL Server 執行個體存取 XML 資料、將資料帶入 .NET 環境、處理資料，以及將更新傳回至 SQL Server 的應用程式。 在 SQL Server 中可透過數種方式來使用 XML 資料，包括資料儲存及用來擷取資料的參數值。 .NET 中的 **SqlXml** 類別支援用戶端在 SQL Server 內，使用儲存在 XML 資料行中的資料。 如需詳細資訊，請參閱 SQL Server 線上叢書中的「SQLXML 受控類別」。

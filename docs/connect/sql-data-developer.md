# <a name="sql-data-developer"></a>SQL 資料開發人員
若要建立資料中心的解決方案，跨行動裝置和桌上型電腦的網頁伺服器、 企業伺服器及雲端使用 Microsoft SQL 資料平台。  

## <a name="sql-data-storage"></a>SQL 資料儲存
* [SQL Server Database Engine](../database-engine/configure-windows/sql-server-database-engine.md)： 使用 SQL Server Database Engine 來建立線上交易處理或線上分析處理資料的關聯式資料庫。 
* [SQL Azure](https://docs.microsoft.com/azure/sql-database/)： 使用 SQL Azure，若要將資料庫移至雲端 
* [SQL Compact](https://www.microsoft.com/download/details.aspx?id=17876): SQL Server Compact 建置獨立和行動裝置、 桌上型電腦、 偶爾連接的應用程式和 Web 用戶端。
* [LocalDB](../database-engine/configure-windows/sql-server-2016-express-localdb.md)： 在開發期間的使用 LocalDB，撰寫和測試而不需要管理完整的伺服器執行個體的 SQL Server 的 TRANSACT-SQL 程式碼。

## <a name="sql-data-tools"></a>SQL 資料工具
* [SQL Server Data Tools ](../ssdt/download-sql-server-data-tools-ssdt.md) ： 使用 SSDT 來建置關聯式資料庫、 Azure SQL database、 Integration Services 封裝、 Analysis Services 資料模型和 Reporting Services 報告。
* [SQL Server 管理工具](../ssms/download-sql-server-management-studio-ssms.md)： 使用 SSMS 設定、 監視及管理 SQL Server 執行個體。

## <a name="sql-data-access"></a>SQL 資料存取
* [SQL 用戶端驅動程式](sql-connection-libraries.md)： 若要連線，使用 SQL 驅動程式查詢、 更新、 插入或刪除 SQL database 中的資料。
* [Entity Framework](https://msdn.microsoft.com/library/gg696172.aspx)： 使用 Entity Framework 來輕鬆地存取資料庫時使用 LINQ，直接存取 SQL Server，此選項可控制 Entity Data Model (EDM) 和未經處理的關聯式結構之間的對應。 
* [Windows Communication Foundation (WCF)](https://msdn.microsoft.com/library/dd456779.aspx)： 使用資料服務，幾乎是全包解決方案來輕鬆地建立和使用網路和內部網路上的 OData 服務
* [同步處理架構](https://msdn.microsoft.com/library/jj839436.aspx)： 使用 Sync Framework，以啟用離線存取任何資料類型、 任何資料存放區、 任何傳輸通訊協定和任何網路拓撲。
* [Reactive Extensions](https://msdn.microsoft.com/library/hh242985.aspx)： 使用 Reactive Extensions (Rx) 來執行事件資料流程式設計和簡化非同步程式設計，使用可預見的序列和 LINQ 樣式查詢運算子。  使用 Reactive Extensions for JavaScript (RxJS)，來簡化非同步的回呼和事件導向程式設計上使用可預見的序列。
* [CLR 整合](../relational-databases/clr-integration/common-language-runtime-clr-integration-programming-concepts.md)：利用 CLR 整合，使用任何 .NET Framework 語言 (包括 Microsoft Visual Basic .NET 及 Microsoft Visual C#) 來撰寫預存程序、觸發程序、使用者定義型別、使用者定義函式、使用者定義彙總及資料流資料表值函式。 
* [SQLXML 4.0](../relational-databases/sqlxml/sqlxml-4-0-programming-concepts.md)： 使用 SQLXML 4.0 中，將關聯式資料匯出至 XML。

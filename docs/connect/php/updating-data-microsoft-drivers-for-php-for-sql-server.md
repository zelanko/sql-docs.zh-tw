# <a name="updating-data-microsoft-drivers-for-php-for-sql-server"></a>更新資料 (Microsoft Drivers for PHP for SQL Server)
[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]

本節的主題說明如何檢查常見使用案例，以更新資料庫中的資料。  
  
使用 [!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)] 來更新資料庫中資料的步驟可以彙總如下：  
  
1.  定義 Transact-SQL 查詢，以執行插入、更新或刪除作業。  
  
2.  更新參數化查詢的參數值。  
  
3.  以更新的參數值執行 Transact-SQL 查詢 (如果適用)。 如需有關執行查詢的詳細資訊，請參閱＜ [Comparing Execution Functions](../../connect/php/comparing-execution-functions.md) ＞。

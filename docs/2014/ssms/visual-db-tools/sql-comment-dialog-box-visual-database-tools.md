# <a name="sql-comment-dialog-box-visual-database-tools"></a>SQL 註解對話方塊 (Visual Database Tools)
  這個對話方塊提供可編輯區域，您可在其中撰寫查詢的詳細說明。 若要存取此對話方塊，請在查詢設計工具和檢視表設計師中開啟查詢或檢視表，然後在 [屬性] 視窗中選取 [SQL 註解]，再按一下 [屬性] 欄位右方的省略符號 (**...**)。

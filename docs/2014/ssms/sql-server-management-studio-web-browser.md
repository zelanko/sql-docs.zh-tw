# <a name="sql-server-management-studio-web-browser"></a>SQL Server Management Studio Web 瀏覽器
  [!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)] 主控了某個版本的 [!INCLUDE[msCoName](../includes/msconame-md.md)] Internet Explorer。 這個 Web 瀏覽器可讓您在不離開 [!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)]的情況下，瀏覽 URL 以及檢視 MSDN Library 說明主題。 您可以在 [檢視]  功能表上，指向 [Web 瀏覽器]  ，再按一下 [顯示瀏覽器]  來存取這個 Web 瀏覽器。

# <a name="show-advanced-options-server-configuration-option"></a>顯示進階選項伺服器組態選項
  **show advanced options** 選項可用來顯示 **sp_configure** 系統預存程序進階選項。 若將 **show advanced options** 設成 1，就可以使用 **sp_configure**列出進階選項。 預設值是 0。  
  
 設定會立即生效，伺服器不必重新啟動。

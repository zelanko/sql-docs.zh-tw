# <a name="compare-files"></a>比較檔案
  您可以比較檔案來判斷檔案如何進入目前的狀態。 例如，如果您在簽入特定來源檔案版本之後，在程式碼專案的某項建置中偵測到錯誤，您可以比較目前的檔案版本和先前的版本。 這可以協助您找出引起錯誤的程式碼。  
  
 您可以利用 [!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)] 來比較專案或檔案的本機副本與原始檔控制所儲存的版本，或比較兩個本機檔案。 此外，使用**歷程記錄**命令時，您可以比較任兩個原始檔控制版本。  
  
### <a name="to-compare-files"></a>比較檔案  
  
1.  在 [方案總管] 中，選取一個專案或某個專案檔。  
  
2.  在上**檔案**功能表上，指向**原始檔控制**，然後按一下**比較**。  
  
3.  在 [**差異選項**] 對話方塊中，選取適當的選項，然後按一下**報表**。

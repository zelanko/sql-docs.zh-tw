# <a name="run-sql-server-profiler"></a>執行 SQL Server Profiler
  若要執行 [!INCLUDE[ssSqlProfiler](../includes/sssqlprofiler-md.md)]，請在 **[開始]** 功能表上，依序指向 **[所有程式]**、 **[Microsoft SQL Server 2014]** 和 **[效能工具]**，然後按一下 **[SQL Server Profiler]**。

# <a name="view-project-history"></a>檢視專案記錄
  [!INCLUDE[msCoName](../includes/msconame-md.md)] Visual SourceSafe (VSS) 專案記錄包括一份在每個專案檔上所採用的完整動作清單，其中包括建立、新增、刪除和復原檔案。  
  
> [!NOTE]  
>  Visual SourceSafe 專案通常稱為原始檔控制伺服器資料夾，它是原始檔控制檔案儲存在伺服器中的伺服器版本的位置。  
  
 您可以利用 [!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)] 來檢查目前載入的方案所屬之 Visual SourceSafe 專案記錄。 您可以根據專案記錄中所顯示的資訊來擷取檔案版本的本機副本、還原已刪除的版本，或在專案之間共用檔案版本。  
  
### <a name="to-view-the-history-of-a-vss-project"></a>檢視 VSS 專案記錄  
  
1.  在 [方案總管] 中，選取一個專案。  
  
2.  在 **檔案**功能表上，指向**原始檔控制**，按一下 **檢視歷程記錄**。  
  
3.  在 **歷程記錄**\<專案 > 對話方塊方塊中，執行下列動作：  
  
    -   檢視原始檔控制系統的所選檔案副本。  
  
    -   檢視所選檔案的詳細資訊。  
  
    -   將所選檔案擷取到本機磁碟中。  
  
    -   簽出所選的檔案。  
  
    -   在兩個原始檔控制專案之間共用所選的檔案。  
  
    -   將記錄報表匯出到印表機、檔案或 [剪貼簿] 中。

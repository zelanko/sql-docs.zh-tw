# <a name="lesson-5-testing-models-basic-data-mining-tutorial"></a>第 5 課：測試模型 (基本資料採礦教學課程)
  現在您已經使用目標郵寄案例定型集處理過模型，所以將會針對測試集來測試模型。 驗證是資料採礦程序中的一個重要步驟。 將模型部署到實際環境之前，了解目標郵寄採礦模型對實際資料的執行效能有多好是非常重要的事情。  
  
 由於測試集中的資料已經包含自行車購買的已知值，所以可以輕鬆地判斷模型的預測是否正確。 [!INCLUDE[ssSampleDBCoFull](../includes/sssampledbcofull-md.md)]行銷部門會使用執行最佳的模型來識別其目標郵寄行銷活動的客戶。  
  
 在這一課，您將使用多種方法來驗證模型：  
  
1.  您將針對測試集進行預測，以查看模型對已知結果的精確度。 您將使用增益*圖*來測量其效益。  
  
     [使用增益圖測試精確度 &#40;基本資料採礦教學課程&#41;](../../2014/tutorials/testing-accuracy-with-lift-charts-basic-data-mining-tutorial.md)  
  
2.  您要針對篩選的資料子集測試模型。 您可以在同一個增益圖中比較多個模型。  
  
     [&#40;基本資料採礦教學課程中測試篩選過的模型&#41;](../../2014/tutorials/testing-a-filtered-model-basic-data-mining-tutorial.md)  
  
 如需模型驗證的一般詳細資訊，請參閱[資料採礦概念](../../2014/analysis-services/data-mining/data-mining-concepts.md)。

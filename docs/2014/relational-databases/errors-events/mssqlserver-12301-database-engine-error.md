# <a name="mssqlserver_12301"></a>MSSQLSERVER_12301
    
## <a name="details"></a>詳細資料  
  
|屬性|值|  
|-|-|  
|產品名稱|SQL Server|  
|事件識別碼|12301|  
|事件來源|MSSQLSERVER|  
|元件|SQLEngine|  
|符號名稱|HK_UNSUPPORTED_NULLABLE_COLUMNS|  
|訊息文字|'*construct*' 不支援索引鍵中可為 Null 的資料行。|  
  
## <a name="user-action"></a>使用者動作

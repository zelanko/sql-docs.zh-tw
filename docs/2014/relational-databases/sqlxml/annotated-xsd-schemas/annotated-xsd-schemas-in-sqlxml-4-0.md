# <a name="annotated-xsd-schemas-in-sqlxml-40"></a>SQLXML 4.0 中的註解式 XSD 結構描述
  本章節提供在 SQLXML 4.0 中使用註解式 XSD 結構描述的相關資訊。 在舊版的 SQLXML 中，XML-Data Reduced (XDR) 結構描述有提供類似的功能。 本章節也提供用於舊版應用程式的 XDR 資訊。  
  
## <a name="in-this-section"></a>本節內容  
 [批註式 XSD 架構簡介 &#40;SQLXML 4.0&#41;](introduction-to-annotated-xsd-schemas-sqlxml-4-0.md)  
 提供 SQLXML 4.0 中註解式 XSD 結構描述的概觀。  
  
 [在 XSD 架構中使用批註 &#40;SQLXML 4.0&#41;](../../sqlxml-annotated-xsd-schemas-using/using-annotations-in-xsd-schemas-sqlxml-4-0.md)  
 描述如何在各種狀況下使用 XSD 結構描述中的註解，並提供範例。  
  
 [在查詢中使用批註式 XSD 架構 &#40;SQLXML 4.0&#41;](using-annotated-xsd-schemas-in-queries-sqlxml-4-0.md)  
 描述如何在查詢中使用註解式 XSD 結構描述。  
  
 [將批註式 XDR 架構轉換為對等的 XSD 架構 &#40;SQLXML 4.0&#41;](converting-annotated-xdr-schemas-to-equivalent-xsd-schemas-sqlxml-4-0.md)  
 描述如何將 XDR 結構描述轉換成 SQLXML 4.0 的同等 XSD 結構描述。  
  
 [批註式 XDR 架構 &#40;在 SQLXML 4.0 中被取代&#41;](annotated-xdr-schemas-deprecated-in-sqlxml-4-0.md)  
 舊版的 SQLXML 支援 XDR 結構描述中的註解。 註解式 XDR 結構描述在 SQLXML 4.0 中已被取代，但是本章節是要當做舊版應用程式的參考。

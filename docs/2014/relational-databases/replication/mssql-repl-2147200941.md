# <a name="mssql_repl-2147200941"></a>MSSQL_REPL-2147200941
    
## <a name="message-details"></a>訊息詳細資料  
  
|||  
|-|-|  
|產品名稱|SQL Server|  
|事件識別碼|-2147200941|  
|事件來源|MSSQLServer|  
|符號名稱||  
|訊息文字|合併處理在評估訂閱者的分割區驗證運算式期間，偵測到不相符的狀況。 可重新初始化訂閱來解決此問題。|  
  
## <a name="explanation"></a>說明  
 合併處理在評估訂閱者的分割區驗證運算式期間，偵測到不相符的狀況。  
  
## <a name="user-action"></a>使用者動作  
 重新初始化這項訂閱。

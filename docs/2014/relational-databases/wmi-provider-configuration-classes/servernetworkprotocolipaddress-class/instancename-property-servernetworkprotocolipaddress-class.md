## <a name="syntax"></a>語法  
  
```  
  
object  
.InstanceName [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 [ServerNetworkProtocolIPAdress 類別](servernetworkprotocolipaddress-class.md)物件。

 指定 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]執行個體名稱的字串值。

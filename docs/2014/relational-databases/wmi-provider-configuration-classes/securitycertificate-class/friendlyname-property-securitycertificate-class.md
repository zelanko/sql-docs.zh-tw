# <a name="friendlyname-property-securitycertificate-class"></a>FriendlyName 屬性 (SecurityCertificate 類別)
  取得或設定安全性憑證的替代易記名稱。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.FriendlyName [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 代表安全性憑證的 [SecurityCertificate 類別](securitycertificate-class.md) 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定安全性憑證之替代易記名稱的字串值。

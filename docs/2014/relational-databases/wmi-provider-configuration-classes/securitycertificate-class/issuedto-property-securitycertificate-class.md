# <a name="issuedto-property-securitycertificate-class"></a>IssuedTo 屬性 (SecurityCertificate 類別)
  取得安全性憑證發行目標的物件。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.IssuedTo [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 代表安全性憑證的 [SecurityCertificate 類別](securitycertificate-class.md) 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定安全性憑證之發行目標物件的字串值。

# <a name="protocolname-property-servernetworkprotocol-class"></a>ProtocolName 屬性 (ServerNetworkProtocol 類別)
  取得伺服器網路通訊協定的名稱。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.ProtocolName [= value]  
```  
  
## <a name="parts"></a>組件  
 *目標*  
 代表實例所使用之[!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]網路通訊協定的[ServerNetworkProtocol 類別](servernetworkprotocol-class.md)物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定伺服器網路通訊協定名稱的字串。

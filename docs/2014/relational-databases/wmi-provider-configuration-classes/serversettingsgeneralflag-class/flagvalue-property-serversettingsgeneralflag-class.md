# <a name="flagvalue-property-serversettingsgeneralflag-class"></a>FlagValue 屬性 (ServerSettingsGeneralFlag 類別)
  取得旗標的值。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.FlagValue [= value]  
```

 *物件*  
 表示伺服器設定之一般旗標的 [ServerSettingsGeneralFlag 類別](serversettingsgeneralflag-class.md) 物件。

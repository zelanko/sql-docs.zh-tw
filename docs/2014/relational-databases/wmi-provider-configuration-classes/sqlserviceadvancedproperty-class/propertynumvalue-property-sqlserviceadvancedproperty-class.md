# <a name="propertynumvalue-property-sqlserviceadvancedproperty-class"></a>PropertyNumValue 屬性 (SqlServiceAdvancedProperty 類別)
  取得進階屬性的數值。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.PropertyNumValue [= value]  
```  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定進階屬性之數值的 uint32 值。

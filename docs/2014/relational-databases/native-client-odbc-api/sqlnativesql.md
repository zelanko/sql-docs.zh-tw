# <a name="sqlnativesql"></a>SQLNativeSql
  Native Client ODBC 驅動程式會滿足 SQLNativeSql 要求，而不需造訪伺服器。 **** [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 此函數可有效率地測試 SQL 陳述式的語法。 語法檢查不會判斷 SQL 語句中的識別碼或運算式結果是否有效，而且[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] **SQLNativeSql**所傳回的原生 SQL 可能無法執行。

# <a name="sqlfetch"></a>SQLFetch
  SQLFetch 支援增強型日期和時間功能。  
  
 如需如何轉換日期/時間類型的結果資料行值的資訊，請參閱[從 SQL 轉換成 C](../native-client-odbc-date-time/datetime-data-type-conversions-from-sql-to-c.md)。  
  
## <a name="sqlfetch-support-for-large-clr-udts"></a>大型 CLR UDT 的 SQLFetch 支援  
 `SQLFetch` 支援大型 CLR 使用者定義型別 (UDT)。 如需詳細資訊，請參閱[Large CLR User-Defined 類型&#40;ODBC&#41;](../native-client/odbc/large-clr-user-defined-types-odbc.md)。

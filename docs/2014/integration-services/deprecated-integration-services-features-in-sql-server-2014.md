# <a name="deprecated-integration-services-features-in-sql-server-2014"></a>SQL Server 2014 Integration Services 已被取代的功能
  本主題描述 [!INCLUDE[ssISnoversion](../includes/ssisnoversion-md.md)] 目前版本中仍然可用但已被取代的功能 [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssISnoversion](../includes/ssisnoversion-md.md)] 。 這些功能將在未來的 [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]版本中移除。 已被取代的功能不應在新應用程式中使用。

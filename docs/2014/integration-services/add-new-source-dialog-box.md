# <a name="add-new-source-dialog-box"></a>加入新來源對話方塊
  下表列出可以在 [加入新來源] 對話方塊中使用的選項。  
  
|選項|描述|  
|------------|-----------------|  
|類型|選取您要連接的來源類型。|  
|連接管理員|選取現有的連線管理員，或按一下 [\<新增>] 以建立新的連線管理員。|  
|僅顯示已安裝|指定是否只要檢視已安裝的來源。|  
|[確定]|按一下以儲存您的變更，並開啟任何後續的對話方塊設定其他選項。|

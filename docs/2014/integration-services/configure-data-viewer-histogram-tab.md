# <a name="configure-data-viewer-histogram-tab"></a>設定資料檢視器 (長條圖索引標籤)
  使用 **[設定資料檢視器]** 對話方塊的 **[長條圖]** 索引標籤，即可指定要顯示的資料行。 長條圖會將來自數值資料行的資料值顯示為數值分佈。 圖表只會反映一個緩衝區的內容。  
  
## <a name="options"></a>選項。  
 **視覺化資料行**  
 選取要視覺化為長條圖的資料行。

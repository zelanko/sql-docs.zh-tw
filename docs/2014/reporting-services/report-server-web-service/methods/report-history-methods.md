# <a name="report-history-methods"></a>報表記錄方法
  您可以使用這些方法建立並管理項目記錄快照集。  
  
|方法|動作|  
|------------|------------|  
|<xref:ReportService2010.ReportingService2010.CreateItemHistorySnapshot%2A>|產生指定之目錄項目的項目記錄快照集。|  
|<xref:ReportService2010.ReportingService2010.DeleteItemHistorySnapshot%2A>|刪除指定之目錄項目的個別項目記錄快照集。|  
|<xref:ReportService2010.ReportingService2010.GetItemHistoryLimit%2A>|傳回指定之目錄項目的項目記錄快照集限制。|  
|<xref:ReportService2010.ReportingService2010.GetItemHistoryOptions%2A>|傳回為目錄項目產生的項目記錄快照集選項設定與屬性。|  
|<xref:ReportService2010.ReportingService2010.ListItemHistory%2A>|傳回指定之目錄項目的項目記錄快照集及其屬性清單。|  
|<xref:ReportService2010.ReportingService2010.SetItemHistoryLimit%2A>|指定報表伺服器所保留的項目快照集數。|  
|<xref:ReportService2010.ReportingService2010.SetItemHistoryOptions%2A>|設定項目記錄選項，以指定何時為指定的目錄項目建立項目記錄快照集。|

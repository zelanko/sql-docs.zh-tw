# <a name="pathname-property-wmi-msreportserver_configurationsetting"></a>PathName 屬性 (WMI MSReportServer_ConfigurationSetting)
  指定報表伺服器執行個體的安裝路徑。 唯讀。  
  
## <a name="syntax"></a>語法  
  
```vb  
public Dim PathName As String  
```  
  
```csharp  
public string PathName;  
```  
  
## <a name="property-values"></a>屬性值  
 代表安裝路徑的 `String` 物件

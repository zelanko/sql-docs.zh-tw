# <a name="unattendedexecutionaccount-property-wmi-msreportserver_configurationsetting"></a>UnattendedExecutionAccount 屬性 (WMI MSReportServer_ConfigurationSetting)
  傳回自動執行報表時，報表伺服器模擬的使用者帳戶。 唯讀。  
  
## <a name="syntax"></a>語法  
  
```vb  
Public Dim UnattendedExecutionAccount As String  
```  
  
```csharp  
public string UnattendedExecutionAccount;  
```  
  
## <a name="property-values"></a>屬性值  
 代表帳戶名稱的 `String` 物件。

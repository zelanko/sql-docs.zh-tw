# <a name="image-text-box-rectangle-line-f1-help"></a>圖像、文字方塊、矩形、線條的 F1 説明
  本節中的主題提供有關選項的詳細資訊，您可在報表設計師中，從對話方塊的影像、文字方塊、矩形和線條上設定這些選項。  
  
 若要從 **[報表屬性]** 對話方塊中存取這些主題，請按一下 **[說明]**。

# <a name="choose-the-deployment-location"></a>選擇部署位置
  使用報表精靈的這個頁面，即可設定報表伺服器專案的部署伺服器和資料夾。  
  
## <a name="options"></a>選項。  
 **報表伺服器**  
 例如，輸入報表伺服器虛擬目錄的 URL http://localhost/reportserver。 這是報表伺服器的虛擬目錄，而非報表管理員。 依預設，報表伺服器在安裝時會建立名稱為 reportserver 的虛擬目錄。  
  
 **部署資料夾**  
 輸入用於發行報表伺服器專案所包含報表的目的資料夾名稱。 此資料夾的預設值是專案名稱。 如果報表伺服器上沒有此資料夾，報表設計師會在發行報表時建立資料夾。

# <a name="report-f1-help"></a>報表 F1 說明
  在本節中的主題提供您可以在報表設定選項的相關資訊**報表屬性** 對話方塊。 若要開啟 [**報告屬性**] 對話方塊中，您在報表設計師中，背景上按一下滑鼠右鍵，然後按一下**報表屬性**。  
  
> [!NOTE]  
>  它很容易使報表的報表主體產生錯誤。 報表主體是包含報表內容的報表元素。 報表包含報表主體，以及選擇性地包含報表頁首及頁尾。 如果您在報表主體上按一下滑鼠右鍵，則無法存取或設定報表屬性。  
  
 若要從 **[報表屬性]** 對話方塊中存取這些主題，請按一下 **[說明]**。

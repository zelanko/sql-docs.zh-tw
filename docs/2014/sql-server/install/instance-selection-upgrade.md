# <a name="instance-selection-upgrade"></a>執行個體選取範圍 (升級)
  使用 [執行個體選取範圍 (升級)] 頁面，即可指定要升級的 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 執行個體。  
  
## <a name="options"></a>選項。  
 您必須同時升級執行個體的所有元件。 您無法在升級期間加入或移除功能。

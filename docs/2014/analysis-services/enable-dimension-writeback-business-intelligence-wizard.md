# <a name="enable-dimension-writeback-business-intelligence-wizard"></a>啟用維度回寫 (商業智慧精靈)
  使用 **[啟用維度回寫]** 頁面，即可啟用或停用選取之維度的回寫功能。  
  
## <a name="options"></a>選項  
 **啟用維度回寫**  
 選取即可讓維度成為可寫入的維度。

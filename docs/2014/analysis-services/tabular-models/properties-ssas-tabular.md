# <a name="properties-ssas-tabular"></a>屬性 (SSAS 表格式)
  [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)] 的表格式模型專案包含各種屬性，可定義專案、模型、報表及部署的行為。 屬性設定會以 XML 格式儲存在 Model.bim 檔案中，但是本節所述的所有屬性都可以在 [!INCLUDE[ssBIDevStudio](../../includes/ssbidevstudio-md.md)] 的 [屬性] 視窗中設定。  
  
## <a name="related-tasks"></a>相關工作  
  
|主題|描述|  
|-----------|-----------------|  
|[Power 檢視報告的屬性&#40;SSAS 表格式&#41;](power-view-reporting-properties-ssas-tabular.md)|本節中的主題針對將連接到 [!INCLUDE[ssCrescent](../../includes/sscrescent-md.md)]和從中進行瀏覽的表格式模型提供說明和組態選項。|  
|[專案屬性&#40;SSAS 表格式&#41;](project-properties-ssas-tabular.md)|提供專案屬性的說明。 專案屬性包含專案檔案和部署選項設定。|  
|[模型屬性的&#40;SSAS 表格式&#41;](model-properties-ssas-tabular.md)|提供模型屬性的說明。 模型屬性會在模型製作期間影響模型專案的建置動作、備份及工作空間資料庫。|  
|[資料表屬性&#40;SSAS 表格式&#41;](table-properties-ssas-tabular.md)|提供基本資料表屬性的說明。 這裡所述的資料表屬性與 [編輯資料表屬性] 對話方塊中的屬性不同，前者會顯示並允許選擇及篩選來源的資料行。|  
|[資料行屬性&#40;SSAS 表格式&#41;](column-properties-ssas-tabular.md)|提供資料行屬性的說明。 資料行屬性會定義資料行的資料類型、格式及隱藏設定。|  
|[設定預設的資料模型和部署屬性&#40;SSAS 表格式&#41;](configure-default-data-modeling-and-deployment-properties-ssas-tabular.md)|提供預設模型和部署屬性的說明及組態步驟。 預設屬性會套用至新的表格式模型專案。 建立專案之後，這些屬性可依據您的需求針對特定模型專案進行變更。|

# <a name="column-width-dialog-box-ssas"></a>資料行寬度對話方塊 (SSAS)
  [資料行寬度]  對話方塊可讓您在模型設計師中指定資料行的寬度。 若要存取此對話方塊，請在資料行上按一下滑鼠右鍵，然後按一下 [資料行寬度] 。 您也可以按兩下資料行右邊緣的格線，讓資料行自動調整為資料行內容的大小。  
  
## <a name="uielement-list"></a>UIElement 清單  
 **資料行寬度**  
 為資料行寬度輸入像素數目。

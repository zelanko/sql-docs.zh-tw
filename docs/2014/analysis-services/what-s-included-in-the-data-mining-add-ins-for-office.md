# <a name="what39s-included-in-the-data-mining-add-ins-for-office"></a>什麼&#39;中的資料採礦 Office 增益集所包含
  當您安裝適用於 Office 2013 的資料採礦增益集時，選取所有的安裝選項就會加入以下的工具列和範本：  
  
-   [適用於 Excel 的資料表分析工具](table-analysis-tools-for-excel.md)  
  
     簡單但強大的工作，可使用 Excel 資料表中的資料。 不需任何自訂設定；演算法會偵測正確的參數，並產生了解模型所需的全部圖表及報表。  
  
     如果您還不熟悉資料採礦，請由此處著手。  
  
-   [適用於 Excel 的資料採礦用戶端&#40;SQL Server 資料採礦增益集&#41;](data-mining-client-for-excel-sql-server-data-mining-add-ins.md)  
  
     此增益集提供您可能需要-從清除和取樣資料，以建立模型、 測試模型精確度以及執行交叉驗證每個資料採礦工作的工具。 您也可以管理模型和製作模型文件，以及從範本建立 DMX 查詢。  
  
     有經驗的資料採礦人員將能盡情存取 SQL Server 資料採礦提供的所有工具，包括模型匯入與匯出、預測查詢和內容查詢。  
  
-   [適用於 Visio 的資料採礦圖形](data-mining-shapes-for-visio.md)  
  
     只要將這些圖形拖曳至 Visio 頁面上，以建立連結到採礦模型的圖表，但可以使用 Visio 來自訂圖表，以建立更具吸引力、更互動的呈現方式。  
  
-   [伺服器組態公用程式&#40;資料採礦適用於 Excel 的增益集&#41;](server-configuration-utility-data-mining-add-ins-for-excel.md)  
  
     建立模型時必須連接到資料採礦伺服器。 如果您需要取得設定的說明，只要按一下**協助**，並遵循每個頁面的 [組態精靈]。 您可以自行修正問題，或由精靈提供可讓您傳送給系統管理員以進行正確設定的資訊。  
  
-   [選擇資料和使用範例資料](choosing-data-for-data-mining.md)  
  
     知道該使用哪些資料及如何據以準備，是資料分析的重要關鍵。 本主題提供您有關如何準備資料以使模型盡可能趨於完善的秘訣。  
  
     如果您不熟悉資料採礦，而且只想要使用工具進行實驗，我們建議您使用與增益集一起安裝的範例資料活頁簿。來看看這項資料也可以查看如何準備資料進行時間序列模型或關聯模型的範例。  
  
     **範例資料**  
  
     範例活頁簿中的資料已經預先格式化，方便您嘗試每一種精靈和工具。  
  
    -   DMAddins_SampleData.xlsx  
  
     範例活頁簿的預設位置取決於您安裝的版本：  
  
    -   \<磁碟機 >: \Program Files\Microsoft SQL Server 2014 DM 增益集  
  
    -   \<drive>:\Program Files(x86)\Microsoft SQL Server 2014 DM add-ins

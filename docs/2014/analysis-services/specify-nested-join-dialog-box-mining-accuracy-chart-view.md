# <a name="specify-nested-join-dialog-box-mining-accuracy-chart-view"></a>指定巢狀聯結對話方塊 (採礦精確度圖表檢視)
  [指定巢狀聯結] 對話方塊會在巢狀資料表和案例資料表之間建立關聯性。 若要開啟 [指定巢狀聯結] 對話方塊，選取巢狀資料表，然後按一下 [修改聯結]。  
  
## <a name="options"></a>選項。  
 **加入關聯性**  
 按一下即可開啟 [建立關聯性] 對話方塊。  
  
 **移除關聯性**  
 按一下即可刪除 [圖表] 窗格中您所選取的關聯性。

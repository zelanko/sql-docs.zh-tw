# <a name="browsing-models-in-excel-sql-server-data-mining-add-ins"></a>在 Excel 中瀏覽模型 (SQL Server 資料採礦增益集)
  ![瀏覽資料採礦功能區中的 [模型] 按鈕](media/dmc-browse.gif "資料採礦功能區中的 瀏覽模型按鈕")  
  
 以視覺效果方式探索模型通常是了解分析所發現之規則和關聯性最簡單快速的方法。 您可以使用適用於 Excel 的資料採礦用戶端來瀏覽在目前 Excel 工作階段期間建立的暫時性模型，以及儲存在 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 執行個體中的模型。  
  
 若要瀏覽模型，您可以從可用模型的清單中選取模型及其關聯的結構，而且此增益集會自動挑選該資料採礦演算法所適合的檢視器。 您可以鑽研到最感興趣的趨勢、篩選已完成的資料採礦模型，並且將圖形和資料複製到 Excel 或 Visio。  
  
## <a name="using-the-browse-model-wizard"></a>使用瀏覽模型精靈  
  
1.  按一下 [**資料採礦**] 索引標籤。  
  
2.  在 **模型使用方式**群組中，按一下**瀏覽**。  
  
3.  在 **選取模型** 對話方塊中，從清單中選擇採礦模型，然後按一下**下一步**。  
  
4.  此精靈會開啟**瀏覽**適用於您所選取的模型類型的視窗。  
  
## <a name="list-of-data-mining-viewers"></a>資料採礦檢視器的清單

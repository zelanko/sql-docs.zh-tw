# <a name="translation-element-assl"></a>Translation 元素 (ASSL)
  提供當地語系化的翻譯的父系[翻譯](../collections/translations-element-assl.md)集合。  
  
## <a name="syntax"></a>語法  
  
```xml  
  
<Translations>  
   <Translation xsi:type="Translation">...</Translation>  
   <!-- or -->  
   <Translation xsi:type="AttributeTranslation">...</Translation>  
</Translations>  
```  
  
## <a name="element-characteristics"></a>元素特性  
  
|特性|描述|  
|--------------------|-----------------|  
|資料類型和長度|[翻譯](../data-type/translation-data-type-assl.md)， [AttributeTranslation](../data-type/attributetranslation-data-type-assl.md)|  
|預設值|None|  
|基數|0-n：出現一次以上的選擇性元素。|  
  
## <a name="element-relationships"></a>元素關聯性  
  
|關聯性|元素|  
|------------------|-------------|  
|父元素|[翻譯](../collections/translations-element-assl.md)|  
|子元素|None|

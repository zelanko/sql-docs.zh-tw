# <a name="dbtablename-element-assl"></a>DbTableName 元素 (ASSL)
  包含父元素所繫結之資料表的名稱。  
  
## <a name="syntax"></a>語法  
  
```xml  
  
<TableBinding> <!-- or TableNotification -->  
   ...  
   <DbTableName>...</DbTableName>  
   ...  
</TableBinding>  
```  
  
## <a name="element-characteristics"></a>元素特性  
  
|特性|描述|  
|--------------------|-----------------|  
|資料類型和長度|String|  
|預設值|無|  
|基數|1-1：只能出現一次的必要元素。|  
  
## <a name="element-relationships"></a>元素關聯性  
  
|關聯性|元素|  
|------------------|-------------|  
|父元素|[TableBinding](../data-type/binding-data-type-assl.md)， [TableNotification](../objects/tablenotification-element-assl.md)|  
|子元素|無|  
  
## <a name="remarks"></a>備註  
 對應至父系的項目`DbTableName`在 「 分析管理物件 (AMO) 物件模型是<xref:Microsoft.AnalysisServices.TableBinding>和<xref:Microsoft.AnalysisServices.TableNotification>。

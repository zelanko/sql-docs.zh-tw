# <a name="deploying-and-scaling-mining-models-data-mining-add-ins-for-excel"></a>部署及調整採礦模型 (適用於 Excel 的資料採礦增益集)
  中的工具**模型使用方式**並**管理**群組可協助您管理和瀏覽現有的採礦模型。 您可以使用這些工具，檢視儲存在 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 執行個體上的任何模型，而不單只限使用增益集建立的模型。  
  
 如果您有必要的權限，可以在不離開 Excel 的情況下，刪除、修改、重新命名或處理現有的採礦模型和結構。  
  
## <a name="model-usage-toolbar"></a>模型使用方式工具列  
  
### <a name="browse"></a>瀏覽  
 使用**瀏覽**來選取現有的資料採礦模型，然後檢視及瀏覽此模型包含多個圖形和工具的檢視器中的精靈。  
  
 如需詳細資訊，請參閱 < [Excel 中的 瀏覽模型&#40;SQL Server 資料採礦增益集&#41;](browsing-models-in-excel-sql-server-data-mining-add-ins.md)。  
  
### <a name="document-model"></a>文件模型  
 按一下 **文件模型**啟動精靈，此精靈所建立的採礦結構和採礦模型所建立的報表。 您可以建立基本報表或進階報表。 報表包含了資料行和模型中繼資料，因此對於記載您的工作及追蹤模型內的變更非常實用。  
  
 如需詳細資訊，請參閱 <<c0> [ 記載的採礦模型&#40;適用於 Excel 的資料採礦增益集&#41;](documenting-mining-models-data-mining-add-ins-for-excel.md)。</c0>  
  
### <a name="query"></a>查詢  
 按一下 **查詢**來啟動**查詢**精靈。 此精靈將以互動方式帶領您逐步完成針對現有資料採礦模型建立預測查詢的程序。  
  
 若要進一步自訂查詢，或建立不包含在精靈中的查詢，只要按一下**進階**按鈕以啟動**資料採礦進階查詢編輯器**。  
  
 如需詳細資訊，請參閱 <<c0> [ 查詢&#40;SQL Server 資料採礦增益集&#41;](query-sql-server-data-mining-add-ins.md)。</c0>  
  
### <a name="data-mining-advanced-query-editor"></a>資料採礦進階查詢編輯器  
 在此編輯器中，您可以使用資料採礦延伸模組 (DMX) 範本著手建立查詢，然後修改輸入、輸出、演算法及參數，以建立自訂採礦模型、採礦結構或預測查詢。  
  
 如需詳細資訊，請參閱 <<c0> [ 進階資料採礦查詢編輯器](advanced-data-mining-query-editor.md)。  
  
## <a name="management"></a>管理  
 使用**管理模型**，檢視目前連接上的現有模型的精靈。 您也可以刪除、重新命名、處理或匯入及匯出採礦模型和結構。  
  
 如需詳細資訊，請參閱 <<c0> [ 管理模型&#40;SQL Server 資料採礦增益集&#41;](manage-models-sql-server-data-mining-add-ins.md)。</c0>

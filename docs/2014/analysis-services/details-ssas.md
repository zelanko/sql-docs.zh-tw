# <a name="details-ssas"></a>詳細資料 (SSAS)
  **[資料表匯入精靈]** 的這個頁面可讓您檢視成功或失敗之資料匯入作業的相關資訊。 若要從 [!INCLUDE[ssBIDevStudio](../includes/ssbidevstudio-md.md)]存取精靈，請按一下 **[模型]** 功能表上的 **[從資料來源匯入]**。

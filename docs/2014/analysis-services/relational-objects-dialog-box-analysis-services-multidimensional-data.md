# <a name="relational-objects-dialog-box-analysis-services---multidimensional-data"></a>關聯式物件對話方塊 (Analysis Services - 多維度資料)
  使用 **中的** [關聯式物件] [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)] 對話方塊，即可從您要設定主動式快取的資料來源選取資料表、檢視或其他物件。 在 [儲存選項][!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)]** 對話方塊的 [通知]**** 索引標籤上，按一下 [指定追蹤資料表]**** 選項中的省略符號按鈕 (**...**)，即可在 ** 中顯示此對話方塊。  
  
## <a name="options"></a>選項。  
 **選取想要設定主動式快取的關聯式物件**  
 選取要追蹤的物件。此選項會顯示可用物件及其類型的方格，其符合 [**篩選**] 中設定的篩選（如果未設定 [**篩選**]，則為所有資料表），且尚未在目前圖表中顯示。  
  
 **Filter**  
 輸入用來限制所列出物件的篩選，然後按一下按鈕來篩選 [選取想要設定主動式快取的關聯式物件]**** 中列出的資料表。  
  
 **顯示系統物件**  
 選取即可在 [選取想要設定主動式快取的關聯式物件]**** 中顯示資料來源的系統物件。

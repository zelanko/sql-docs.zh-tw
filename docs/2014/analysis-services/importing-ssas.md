# <a name="importing-ssas"></a>匯入 (SSAS)
  [資料表匯入精靈]**** 的這個頁面可讓您檢視匯入作業的進度。 若要從 [!INCLUDE[ssBIDevStudio](../includes/ssbidevstudio-md.md)]存取精靈，請按一下 **[模型]** 功能表上的 **[從資料來源匯入]**。  
  
## <a name="uielement-list"></a>UIElement 清單  
 **詳細資料**  
 針對每一個資料匯入作業顯示以下資訊。  
  
|資料行|描述|  
|------------|-----------------|  
|**工作專案**|顯示正在匯入之資料表或檢視表的名稱。|  
|**狀態**|指出是否已成功匯入資料表或檢視表以及匯入的資料列數目。|  
|**訊息**|如果資料表或檢視表匯入失敗，則會顯示詳細資訊的連結。 這項資訊會在 [詳細資料] 視窗中顯示。<br /><br /> 若要嘗試重新匯入資料表或檢視表，請結束精靈，然後重新執行。|  
  
 **停止匯入**  
 按一下即可在完成前停止匯入作業。 已匯入的資料表和檢視都會出現在 [!INCLUDE[ssBIDevStudio](../includes/ssbidevstudio-md.md)] 設計工具中。 尚未匯入的資料表與檢視則不會匯入。

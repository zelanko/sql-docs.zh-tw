# <a name="xquery-expressions"></a>XQuery 運算式
[!INCLUDE[sqlserver](../includes/applies-to-version/sqlserver.md)]

  描述 XQuery 運算式的主題。  
  

 [&#40;XQuery&#41;的主要運算式](../xquery/primary-expressions-xquery.md)  
 描述 XQuery 的主要運算式。 這些包含常值、變數參考、內容項目運算式、建構函式以及函數呼叫。  
  
 [&#40;XQuery&#41;的路徑運算式](../xquery/path-expressions-xquery.md)  
 描述 XQuery 路徑運算式。 這些會找出文件中的節點，例如元素、屬性及文字。  
  
 [&#40;XQuery&#41;的順序運算式](../xquery/sequence-expressions-xquery.md)  
 描述與數字順序搭配使用的 XQuery 運算子。  
  
 [&#40;XQuery&#41;的算術運算式](../xquery/arithmetic-expressions-xquery.md)  
 描述使用 XQuery 中的算術運算式。  
  
 [&#40;XQuery&#41;的比較運算式](../xquery/comparison-expressions-xquery.md)  
 描述 XQuery 中支援的比較運算式。 也就是一般的值、節點比較以及節點類型比較運算式。  
  
 [&#40;XQuery&#41;的邏輯運算式](../xquery/logical-expressions-xquery.md)  
 描述邏輯**and**和**Or**運算子的 XQuery 支援。  
  
 [&#40;XQuery&#41;的 XML 結構](../xquery/xml-construction-xquery.md)  
 描述 XQuery 建構函式，以允許您建構查詢中的 XML。  
  
 [FLWOR 語句和反復專案 &#40;XQuery&#41;](../xquery/flwor-statement-and-iteration-xquery.md)  
 描述 FLOWR 反覆運算語法。 這代表 FOR、LET、WHERE、ORDER BY 以及 RETURN。 不支援 LET。  
  
 [&#40;XQuery&#41;的已排序和未排序運算式](../xquery/ordered-and-unordered-expressions-xquery.md)  
 描述 XQuery 作業的排序模式。 根據預設，排序模式會設定為 [已**排序**]。  
  
 [&#40;XQuery&#41;的條件運算式](../xquery/conditional-expressions-xquery.md)  
 描述條件**式 if-then**語句的 XQuery 支援。  
  
 [&#40;XQuery&#41;的量化運算式](../xquery/quantified-expressions-xquery.md)  
 描述 XQuery 中的存在和通用數量詞。  
  
 [&#40;XQuery&#41;的 SequenceType 運算式](../xquery/sequencetype-expressions-xquery.md)  
 描述 XQuery 中的順序類型語法。  
  
 [&#40;XQuery&#41;驗證運算式](../xquery/validate-expressions-xquery.md)  
 不支援**驗證**運算式。

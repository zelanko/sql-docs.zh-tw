# <a name="backup-checksum-default"></a>備份總和檢查碼預設
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]

  使用備份總和檢查碼預設設定，在備份及還原期間啟用或停用備份總和檢查碼。  
  
 下表說明可用的值：  
  
|||  
|-|-|  
|**值**|**意義**|  
|0|已停用。 這是預設值。|  
|1|已啟用|  
  
 這項設定會立即生效。

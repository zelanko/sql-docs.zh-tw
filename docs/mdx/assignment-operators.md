# <a name="assignment-operators"></a>指派運算子


  在多維度運算式 (MDX) 中，指派運算子為等號 (=)。  
  
 指派運算子是用來將值指派給 MDX 指令碼中的 Subcube。 如需詳細資訊，請參閱 [MDX 指令碼基礎觀念 &#40;Analysis Services&#41;](/analysis-services/multidimensional-models/mdx/mdx-scripting-fundamentals-analysis-services)。

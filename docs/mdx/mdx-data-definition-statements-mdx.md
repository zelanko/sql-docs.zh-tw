# <a name="mdx-data-definition-statements-mdx"></a>MDX 資料定義陳述式 (MDX)


  在多維度運算式 (MDX) 中，資料定義陳述式可建立、卸除和管理多維度物件。 下表列出可用的資料定義陳述式。  
  
## <a name="in-this-section"></a>本節內容  
  
|主題|描述|  
|-----------|-----------------|  
|[ALTER CUBE 語句 &#40;MDX&#41;](../mdx/mdx-data-definition-alter-cube.md)|改變指定 Cube 的結構。|  
|[&#40;MDX&#41;建立動作語句](../mdx/mdx-data-definition-create-action.md)|建立與 Cube、維度、階層或從屬物件關聯的動作。|  
|[建立 &#40;MDX&#41;的資料格計算語句](../mdx/mdx-data-definition-create-cell-calculation.md)|在 Cube 內指定的 Tuple 集合上，建立評估 MDX 運算式的計算。|  
|[CREATE GLOBAL CUBE 語句 &#40;MDX&#41;](../mdx/mdx-data-definition-create-global-cube.md)|根據伺服器上 Cube 的 Subcube，建立和擴展本機保存的 Cube。 連接到本機保存的 Cube 不需要連接伺服器。|  
|[&#40;MDX&#41;的 CREATE MEMBER 語句](../mdx/mdx-data-definition-create-member.md)|建立導出成員。|  
|[&#40;MDX&#41;建立會話 CUBE 語句](../mdx/mdx-data-definition-create-session-cube.md)|根據伺服器上的 Cube，建立並擴展可供同一工作階段中所有查詢使用的 Cube。|  
|[&#40;MDX&#41;建立 SET 語句](../mdx/mdx-data-definition-create-set.md)|建立指定 Cube 的命名集。|  
|[建立 &#40;MDX&#41;的子集語句](../mdx/mdx-data-definition-create-subcube.md)|將指定 Cube 或 Subcube 的 Cube 空間重新定義為指定的 Subcube。|  
|[DROP ACTION 語句 &#40;MDX&#41;](../mdx/mdx-data-definition-drop-action.md)|刪除指定 Cube 的指定動作。|  
|[DROP CELL CALCULATION 語句 &#40;MDX&#41;](../mdx/mdx-data-definition-drop-cell-calculation.md)|移除指定的資料格計算。|  
|[DROP MEMBER 語句 &#40;MDX&#41;](../mdx/mdx-data-definition-drop-member.md)|移除導出成員。|  
|[&#40;MDX&#41;的 DROP SET 語句](../mdx/mdx-data-definition-drop-set.md)|移除命名集。|  
|[&#40;MDX&#41;的 DROP 子資料集語句](../mdx/mdx-data-definition-drop-subcube.md)|卸除指定的 Subcube，還原為先前定義的 Cube 或具有指定名稱的 Subcube 定義。|  
|[&#40;MDX&#41;的 REFRESH CUBE 語句](../mdx/mdx-data-definition-refresh-cube.md)|重新整理 Cube 的用戶端快取。|

# <a name="system-databases-in-parallel-data-warehouse-pdw"></a>平行處理資料倉儲（PDW）中的系統資料庫
系統資料庫是 SQL Server PDW 用來管理使用者、資料庫和暫存資料的資料庫。 SQL Server PDW 包括下列系統資料庫： master 和 tempdb。  
  
## <a name="system-database-topics"></a>系統資料庫主題

-   [tempdb 資料庫](tempdb-database.md)

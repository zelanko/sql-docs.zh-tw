# <a name="understanding-admin-console-alerts-in-analytics-platform-system"></a>瞭解 Analytics Platform System 中的管理主控台警示
警示會出現在設備 **管理主控台** 和 **System Center Operations Manager** (SCOM) 中。 使用此警示清單來協助識別哪些警示需要額外的調查。  
  
如需使用 Internet Explorer 連接到 **管理主控台** 的相關資訊，請參閱 [使用管理主控台 &#40;Analytics Platform System&#41;來監視設備 ](monitor-the-appliance-by-using-the-admin-console.md)。 如需 **SCOM**的詳細資訊，請參閱 [使用 System Center Operations Manager &#40;Analytics Platform System 監視設備&#41;](monitor-the-appliance-by-using-system-center-operations-manager.md)  
  
如需使用 Transact-sql 取得警示資訊的相關資訊 \- ，請參閱 [使用系統檢視 &#40;Analytics Platform system&#41;來監視設備 ](monitor-the-appliance-by-using-system-views.md)。  
  
## <a name="types-of-alerts"></a>警示的類型  
指出正常狀態的警示名稱通常不需要調查。 包含文字 NON_CRITICAL 的警示名稱有時需要採取動作。 所有其他類型的警示都需要調查。

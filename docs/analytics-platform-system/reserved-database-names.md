# <a name="reserved-database-names"></a>保留的資料庫名稱
保留的資料庫名稱是無法指派給 SQL Server PDW 中的使用者資料庫的名稱。  
  
保留的資料庫名稱清單：  
  
-   master  
  
-   model  
  
-   msdb  
  
-   tempdb  
  
-   pdwtempdb  
  
-   pdwtempdb1  
  
-   pdwtempdb2  
  
-   : DWConfiguration  
  
-   DWDiagnostics  
  
-   DWQueue

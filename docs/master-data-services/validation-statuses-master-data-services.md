# <a name="validation-statuses-master-data-services"></a>驗證狀態 (Master Data Services)
  在 **[版本管理]** 功能區域的 **[驗證版本]** 頁面上，可能會出現下列狀態。  
  
|狀態|說明|  
|------------|-----------------|  
|等候驗證|正在等候驗證的所有新建成員。|  
|等候重新驗證|正在等候重新驗證的現有成員。|  
|驗證成功|已通過商務規則驗證的成員。|  
|驗證失敗|未通過商務規則驗證的成員。|  
|等候相依成員重新驗證|連同子成員正在等候重新驗證的更新合併成員。|

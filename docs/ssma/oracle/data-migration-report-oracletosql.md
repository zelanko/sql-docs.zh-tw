# <a name="data-migration-report--oracletosql"></a>資料移轉報告 (OracleToSQL)
將資料移轉至之後，[**資料移轉報表**] 對話方塊隨即出現 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 。  
  
## <a name="options"></a>選項。  
**狀態**  
顯示從來源到目標資料庫的資料移轉狀態。  
  
**From**  
來源資料表。  
  
**若要**  
目標資料表。  
  
**資料列總數**  
來源資料表中的資料列數目。  
  
**成功遷移的資料列數目**  
已成功遷移至目標資料表的資料列數目。  
  
**外觀比例**  
已成功遷移資料列的百分比。  
  
**詳細資料**  
如果任何資料移轉失敗，請按一下以顯示報表中選取之資料列的遷移詳細資料。 SSMA 會顯示失敗的原因。  
  
**儲存報表**  
將報表儲存至。CSV， (逗號分隔值) 檔案，您可以使用 Microsoft Excel 來檢查該檔案。

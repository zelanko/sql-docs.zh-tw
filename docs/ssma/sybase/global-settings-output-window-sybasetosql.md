# <a name="global-settings-output-window--sybasetosql"></a>全域設定 （[輸出] 視窗） (SybaseToSQL)
使用**通用設定**對話方塊來指定 SSMA 的設定。  
  
若要存取 輸出 視窗設定 中，在**工具**功能表上，選取**通用設定**，按一下  **GUI**底部的左的窗格，然後選取**輸出 視窗**。  
  
## <a name="options"></a>選項。  
**顯示錯誤清單 窗格**  
選取**True**如果您想要有訊息時顯示錯誤清單 窗格的 SSMA。  
  
**顯示 [輸出] 窗格**  
選取**True**您是否有訊息時，顯示 [輸出] 窗格的 SSMA。

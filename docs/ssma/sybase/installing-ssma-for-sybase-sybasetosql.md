# <a name="installing-ssma--for-sybase-sybasetosql"></a>安裝的 SSMA for Sybase (SybaseToSQL)
[!INCLUDE[msCoName](../../includes/msconame_md.md)][!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)]包含您用來執行從 Sybase Adaptive Server Enterprise (ASE) 來移轉用戶端應用程式移轉小幫手 (SSMA) 的 Sybase[!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)]或 SQL Azure。 它也包含支援資料移轉和 ASE 系統函式的使用在您移轉之資料庫中的延伸模組組件。  
  
您將會執行移轉步驟在電腦上安裝用戶端應用程式。 您必須安裝在正在執行的電腦上的延伸模組組件檔案[!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)]移轉之資料庫裝載的位置。  
  
## <a name="upgrading-ssma-for-sybase"></a>升級的 SSMA for Sybase  
如果您想要將升級至更新版本的 SSMA for Sybase，您必須先解除安裝用戶端和伺服器延伸模組組件，然後再安裝較新版本。  
  
如果您開啟的專案從舊版的 SSMA for Sybase，SSMA 會要求您如果要將專案轉換成較新版本。 您必須按一下**是**使用較新版本的 SSMA 專案。

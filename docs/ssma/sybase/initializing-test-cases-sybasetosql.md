# <a name="initializing-test-cases-sybasetosql"></a>初始化測試案例 (SybaseToSQL)
在這個步驟中，您會指定新的測試案例的初始資訊。  
  
## <a name="parameters"></a>參數  
**測試案例名稱**  
輸入的名稱來識別測試案例。  
  
**建立日期**  
今天的目前日期，自動定義。  
  
**上次修改日期**  
自動; 填入不會變更。  
  
**測試案例描述**  
輸入任何其他資訊來識別測試案例的目的。

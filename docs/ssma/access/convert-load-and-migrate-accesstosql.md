# <a name="convert-load-and-migrate-accesstosql"></a>轉換、載入和遷移（AccessToSQL）
當您使用 [轉換]、[載入] 和 [遷移] 按鈕或 [遷移嚮導] 時，[**轉換]、[載入] 和 [遷移**] 對話方塊會報告遷移狀態。  
  
完成遷移之後，您可以按一下每個區段中的連結，來查看資訊、警告和錯誤訊息。 此連結會顯示有多少錯誤、警告和參考用訊息，例如「0個錯誤、4個警告」。  
  
頁面的 [**遷移所選物件的資料**] 區段也包含 [**報表**] 按鈕。 您可以按一下此按鈕來查看資料移轉報表。  
  
## <a name="options"></a>選項。  
**Report**  
顯示相關聯的轉換或遷移報表。

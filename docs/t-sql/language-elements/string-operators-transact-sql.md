# <a name="string-operators-transact-sql"></a>字串運算子 (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2012-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2012-xxxx-xxxx-xxx-md.md)]

[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 提供下列字串運算子。 字串串連運算子可將兩個以上的下列資料類型合併成一個運算式： 
* 字元或二進位字串
* 資料行 
* 字串和資料行名稱的組合 

在字串比較作業中，萬用字元字串運算子可以比對一或多個字元。 例如，LIKE 和 PATINDEX 是其中兩個作業的範例。

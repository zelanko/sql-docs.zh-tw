# <a name="delete-a-solution"></a>刪除方案
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] 永久刪除解決方案，會從實體儲存體中清除檔案，也會從解決方案中清除檔案項目。  
  
> [!NOTE]  
> 相較於專案和項目，您只能永久刪除方案，無法暫時移除它。  
  
### <a name="to-delete-a-solution"></a>刪除方案  
  
1.  在 [方案總管] 中，選取您要刪除的方案。  
  
2.  按一下 [ **檔案** ] 功能表上的 [ **關閉方案**]。  
  
3.  在 [!INCLUDE[msCoName](../../includes/msconame_md.md)] Windows 的 [檔案總管] 中，尋找和選取您要刪除之方案的相關檔案。  
  
4.  在 [檔案] 功能表上，按一下 [刪除]。

# <a name="add-objects"></a>加入物件
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
限制物件選項對話方塊為僅顯示您想看到的物件類型。  
  
## <a name="options"></a>選項。  
**特定物件**  
開啟 [選取物件]  對話方塊，以進一步定義物件搜尋。  
  
**下列類型的所有物件**  
開啟 [選取物件類型]  對話方塊，以指定應包含在基礎清單中的物件類型。  
  
伺服器  *<servername>*  
在指定的伺服器上加入所有適當的物件。 只有從伺服器範圍對話方塊，例如 [登入 - 新增]  對話方塊開啟時才會出現。  
  
**屬於結構描述的所有物件**  
將 [結構描述名稱]  方塊中所指定之結構描述所擁有的所有物件加入。 僅顯示於資料庫範圍對話方塊，例如 [資料庫使用者 - 新增]  對話方塊。

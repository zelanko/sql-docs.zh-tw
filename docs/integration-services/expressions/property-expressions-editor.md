# 屬性運算式編輯器
  使用 **[屬性運算式編輯器]** 對話方塊，即可建立、編輯或刪除用來更新屬性值的屬性運算式。  
  
## 選項。  
 **屬性**  
 從清單中選取物件屬性。  
  
 **運算式**  
 鍵入常值或運算式，即可更新屬性的值。  
  
> [!NOTE]  
>  使用者定義變數和系統變數的名稱會區分大小寫。  
  
 **運算式 (…)**  
 按一下省略符號，即可開啟 **[運算式產生器]** 對話方塊。 如需詳細資訊，請參閱 [Expression Builder](../../integration-services/expressions/expression-builder.md)。  
  
 **Delete**  
 選取屬性，然後按一下 [刪除]。

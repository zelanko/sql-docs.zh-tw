# <a name="connect-to-data-sources-with-the-sql-server-import-and-export-wizard"></a>連接到資料來源的 SQL Server 匯入和匯出精靈
此章節的主題會示範如何連接到許多常用的資料來源，當您執行的 SQL Server 匯入和匯出精靈 」。 您必須提供資料來源連接資訊在**選擇資料來源**和**選擇目的地**精靈頁面。

本章節的主題描述如何只以**連接到資料來源**從**選擇資料來源**和**選擇目的地**精靈頁面。 如果您想要尋求其他項目，請參閱[相關的工作和內容](#related)。

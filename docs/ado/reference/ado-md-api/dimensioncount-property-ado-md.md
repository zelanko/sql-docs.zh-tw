# <a name="dimensioncount-property-ado-md"></a>DimensionCount 屬性 (ADO MD)
表示 [軸](./axis-object-ado-md.md)上的維度數目。  
  
## <a name="return-values"></a>傳回值  
 傳回 **長** 整數，而且是唯讀的。

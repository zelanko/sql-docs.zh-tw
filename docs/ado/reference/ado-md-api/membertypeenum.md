# <a name="membertypeenum"></a>MemberTypeEnum
指定的設定[型別](../../../ado/reference/ado-md-api/type-property-ado-md.md)屬性[成員](../../../ado/reference/ado-md-api/member-object-ado-md.md)物件。  
  
|常數|值|描述|  
|--------------|-----------|-----------------|  
|**adMemberAll**|4|指出**成員**物件所表示的層級的所有成員。|  
|**adMemberFormula**|3|指出**成員**物件也會使用計算公式的運算式。|  
|**adMemberMeasure**|2|指出**成員**物件所屬的量值維度，並代表數量的屬性。|  
|**adMemberRegular**|1|預設值。 指出**成員**物件都代表商務實體的執行個體。|  
|**adMemberUnknown**|0|無法判斷成員的型別。|

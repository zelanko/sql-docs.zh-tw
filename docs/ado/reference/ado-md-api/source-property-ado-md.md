# <a name="source-property-ado-md"></a>Source 屬性 (ADO MD)
指示中的資料來源[資料格集](../../../ado/reference/ado-md-api/cellset-object-ado-md.md)。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定或傳回**Variant**，並關閉為讀取/寫入[資料格集](../../../ado/reference/ado-md-api/cellset-object-ado-md.md)物件且為唯讀狀態開啟**Cellset**物件。 **Variant**應該包含有效**字串**，例如，MDX 查詢。

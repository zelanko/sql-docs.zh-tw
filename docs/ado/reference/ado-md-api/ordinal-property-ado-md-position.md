# <a name="ordinal-property-ado-md-position"></a>序數屬性 （ADO MD 位置）
可唯一識別[位置](../../../ado/reference/ado-md-api/position-object-ado-md.md)沿座標軸。  
  
## <a name="return-values"></a>傳回值  
 傳回**長**整數和處於唯讀狀態。  
  
## <a name="remarks"></a>備註  
 **序數**屬性[位置](../../../ado/reference/ado-md-api/position-object-ado-md.md)物件會對應至索引**位置**內[位置](../../../ado/reference/ado-md-api/positions-collection-ado-md.md)集合。  
  
 資料格可以快速地擷取使用**序數**的**位置**沿著每個座標軸[項目](../../../ado/reference/ado-md-api/item-property-ado-md-cellset.md)屬性[資料格集](../../../ado/reference/ado-md-api/cellset-object-ado-md.md)物件.

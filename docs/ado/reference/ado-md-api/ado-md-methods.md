# <a name="ado-md-methods"></a>ADO MD 方法

|方法|描述|  
|-|-|  
|[關閉](./close-method-ado-md.md)|關閉開啟的儲存格。|  
|[GetSchemaObject](./getschemaobject-method-ado-md.md)|依唯一名稱， (維度、階層、層級或成員) 來抓取 ADO MD 的架構物件。|  
|[開啟](./open-method-ado-md.md)|抓取多維度查詢的結果，並將結果傳回至儲存格集。|

# <a name="precision-property-adox"></a>有效位數屬性 (ADOX)
表示中的資料值的最大有效位數[資料行](../../../ado/reference/adox-api/column-object-adox.md)。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定並傳回**長**的資料行中的資料值的最大有效位數的值時[類型](../../../ado/reference/adox-api/type-property-column-adox.md)屬性都是數值類型。 **有效位數**會忽略所有其他資料類型。  
  
## <a name="remarks"></a>備註  
 預設值為零 (**0**)。  
  
 這個屬性是唯讀的[資料行](../../../ado/reference/adox-api/column-object-adox.md)物件已經附加至集合。

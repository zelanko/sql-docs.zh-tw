# <a name="keytypeenum"></a>KeyTypeEnum
指定的型別[金鑰](../../../ado/reference/adox-api/key-object-adox.md)： 主要、 外部索引鍵，或唯一。  
  
|常數|值|描述|  
|--------------|-----------|-----------------|  
|**adKeyPrimary**|1|預設值。 索引鍵是主索引鍵。|  
|**adKeyForeign**|2|索引鍵是外部索引鍵。|  
|**adKeyUnique**|3|索引鍵是唯一的。|

# <a name="definedsize-property-adox"></a>DefinedSize 屬性 (ADOX)
表示資料行的指定的大小上限。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定並傳回**長**是以字元為單位的資料值的長度上限的值。  
  
## <a name="remarks"></a>備註  
 預設值為零 (0)。  
  
 這個屬性是唯讀的[資料行](../../../ado/reference/adox-api/column-object-adox.md)物件已經附加至集合。

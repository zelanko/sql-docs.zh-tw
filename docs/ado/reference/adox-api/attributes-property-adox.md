# <a name="attributes-property-adox"></a>屬性的內容 (ADOX)
描述資料行的特性。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定或傳回**長**值。 值，指定所表示之資料表的特性[資料行](../../../ado/reference/adox-api/column-object-adox.md)物件。 值可以是組合[ColumnAttributesEnum](../../../ado/reference/adox-api/columnattributesenum.md)常數。 預設值為零 (**0**)，這兩者都不是**adColFixed**也**adColNullable**。

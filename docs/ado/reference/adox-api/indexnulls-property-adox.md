# <a name="indexnulls-property-adox"></a>IndexNulls 屬性 (ADOX)
指示其索引欄位中具有 null 值的記錄是否有索引項目。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定並傳回[AllowNullsEnum](../../../ado/reference/adox-api/allownullsenum.md)值。 預設值是**adIndexNullsDisallow**。  
  
## <a name="remarks"></a>備註  
 這個屬性是唯讀[索引](../../../ado/reference/adox-api/index-object-adox.md)物件已經附加至集合。

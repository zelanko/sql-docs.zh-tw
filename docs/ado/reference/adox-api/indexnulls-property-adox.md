# <a name="indexnulls-property-adox"></a>IndexNulls 屬性 (ADOX)
指出索引欄位中具有 null 值的記錄是否有索引項目。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定並傳回 [AllowNullsEnum](./allownullsenum.md) 值。 預設值為 **adIndexNullsDisallow**。  
  
## <a name="remarks"></a>備註  
 在已附加至集合的 [索引](./index-object-adox.md) 物件上，這個屬性是唯讀的。  
  
## <a name="applies-to"></a>套用至  
 [Index 物件 (ADOX)](./index-object-adox.md)

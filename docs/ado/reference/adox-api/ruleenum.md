# <a name="ruleenum"></a>RuleEnum
指定刪除 [金鑰](../../../ado/reference/adox-api/key-object-adox.md) 時要遵循的規則。  
  
|常數|值|描述|  
|--------------|-----------|-----------------|  
|**adRICascade**|1|串聯變更。|  
|**adRINone**|0|預設值。 不採取任何動作。|  
|**adRISetDefault**|3|外鍵值設定為預設值。|  
|**adRISetNull**|2|外鍵值設定為 null。|

# <a name="columnattributesenum"></a>ColumnAttributesEnum
指定資料[行](../../../ado/reference/adox-api/column-object-adox.md)的特性。  
  
|持續性|值|描述|  
|--------------|-----------|-----------------|  
|**adColFixed**|1|資料行是固定長度。|  
|**adColNullable**|2|此資料行可能包含 null 值。|

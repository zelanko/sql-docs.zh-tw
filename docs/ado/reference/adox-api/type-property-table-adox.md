# <a name="type-property-table-adox"></a>型別屬性 （資料表） (ADOX)
指出資料表類型。  
  
## <a name="return-values"></a>傳回值  
 傳回**字串**值，指定的類型; 例如"TABLE"、"系統 TABLE"或"全域暫存 」 的資料表。  
  
## <a name="remarks"></a>備註  
 此屬性是唯讀的。

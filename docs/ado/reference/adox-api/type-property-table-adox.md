# <a name="type-property-table-adox"></a>Type 屬性 (Table) (ADOX)
表示資料表的類型。  
  
## <a name="return-values"></a>傳回值  
 傳回指定資料表類型的**字串**值。例如，「資料表」、「系統資料表」或「全域臨時」。  
  
## <a name="remarks"></a>備註  
 這個屬性是唯讀的。

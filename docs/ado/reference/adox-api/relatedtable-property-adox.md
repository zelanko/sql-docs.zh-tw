# <a name="relatedtable-property-adox"></a>RelatedTable 屬性 (ADOX)
指出相關資料表的名稱。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定，並傳回**字串**值。  
  
## <a name="remarks"></a>備註  
 預設值為空字串 ("")。  
  
 如果索引鍵的外部索引鍵，然後**RelatedTable**是包含索引鍵的資料表名稱。

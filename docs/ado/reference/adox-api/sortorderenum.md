# <a name="sortorderenum"></a>SortOrderEnum
指定索引資料行的排序次序。  
  
|持續性|值|描述|  
|--------------|-----------|-----------------|  
|**adSortAscending**|1|預設值。 資料行的排序次序是遞增的。|  
|**adSortDescending**|2|資料行的排序次序為遞減。|  
  
## <a name="applies-to"></a>套用至  
 [SortOrder 屬性 (ADOX)](./sortorder-property-adox.md)

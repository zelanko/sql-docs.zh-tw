# <a name="sortorder-property-adox"></a>SortOrder 屬性 (ADOX)
指出資料行 （僅限索引欄） 的排序順序。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定，並傳回**長**值，可以是其中一個[SortOrderEnum](../../../ado/reference/adox-api/sortorderenum.md)常數。 預設值是**adSortAscending**。  
  
## <a name="remarks"></a>備註  
 此屬性只適用於[資料行](../../../ado/reference/adox-api/column-object-adox.md)中的物件[資料行](../../../ado/reference/adox-api/columns-collection-adox.md)集合[索引](../../../ado/reference/adox-api/index-object-adox.md)。  
  
## <a name="applies-to"></a>適用於

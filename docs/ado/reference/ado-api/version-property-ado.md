# <a name="version-property-ado"></a>Version 屬性 (ADO)
表示 ADO 版本號碼。  
  
## <a name="return-value"></a>傳回值  
 傳回表示版本的 **字串** 值。  
  
## <a name="remarks"></a>備註  
 使用 **version** 屬性可傳回 ADO 執行的版本號碼。  
  
 提供者的版本將以 [屬性](./properties-collection-ado.md) 集合中的動態屬性形式提供。  
  
## <a name="applies-to"></a>套用至

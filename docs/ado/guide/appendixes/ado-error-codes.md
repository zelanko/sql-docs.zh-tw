# <a name="capture-ado-error-codes"></a>擷取 ADO 錯誤代碼
除了提供者錯誤中傳回[錯誤](../../../ado/reference/ado-api/error-object.md)物件[錯誤](../../../ado/reference/ado-api/errors-collection-ado.md)集合本身 ADO 可以將錯誤傳回給執行階段環境的例外狀況處理機制。 使用錯誤截取機制您的程式語言，例如**On Error** Microsoft® Visual Basic 中，陳述式或**try catch** microsoft，以擷取 ADO 錯誤中的區塊。

 ADO 錯誤碼的清單，請參閱[ErrorValueEnum](../../../ado/reference/ado-api/errorvalueenum.md)。

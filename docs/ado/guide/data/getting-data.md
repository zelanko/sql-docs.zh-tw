# <a name="getting-data"></a>取得資料
[ADO 基本概念](../../../ado/guide/data/ado-fundamentals.md)，而[HelloData](../../../ado/guide/data/hellodata-a-simple-ado-application.md)範例將特別介紹建立 ADO 應用程式所需的四個主要作業： 取得資料、 檢查資料、 編輯資料，以及更新資料。 本章節會討論更多詳細資料。  
  
 基本的層級中，有幾個 ADO 物件參與取得資料的作業。 首先您必須連接到資料來源使用 ADO**連線**物件。 然後您傳遞給使用 ADO 的資料來源的指示**命令**物件。 最後，您最常收到的資料在 ADO**資料錄集**物件。  
  
 此章節包含下列主題。  
  
-   [連接到資料來源](../../../ado/guide/data/connecting-to-data-sources.md)  
  
-   [準備和執行命令](../../../ado/guide/data/preparing-and-executing-commands.md)  
  
-   [接收結果](../../../ado/guide/data/receiving-results.md)

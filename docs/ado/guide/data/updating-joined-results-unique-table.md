# <a name="updating-joined-results-unique-table"></a>更新 JOINed 結果：Unique Table
ADO 可讓您以緊密控制特定基底資料表中使用唯一資料表的動態屬性的多個基底資料表上的聯結作業所形成的資料錄集的修改。 如需使用唯一資料表的詳細資訊，請參閱 ADO 程式設計人員參考主題的唯一資料表並更新 Resynch 的動態屬性。

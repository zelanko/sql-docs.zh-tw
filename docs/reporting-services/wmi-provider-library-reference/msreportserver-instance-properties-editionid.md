# <a name="msreportserverinstance-properties---editionid"></a>MSReportServer_Instance 屬性 - EditionID
  傳回報表伺服器的版本。 唯讀。  
  
## <a name="syntax"></a>語法  
  
```vb  
Public Dim EditionID As UInt  
```  
  
```csharp  
public uint EditionID;  
```  
  
## <a name="property-value"></a>屬性值  
 包含報表伺服器版本的 **uint** 。

# <a name="configurationsetting-property---machineaccountidentity"></a>ConfigurationSetting 屬性 - MachineAccountIdentity
  取得安裝報表伺服器的電腦帳戶識別。  
  
## <a name="syntax"></a>語法  
  
```vb  
Public Dim MachineAccountIdentity As String  
```  
  
```csharp  
public string MachineAccountIdentity;  
```  
  
## <a name="property-values"></a>屬性值  
 **String** 值，包含安裝報表伺服器的電腦帳戶識別。

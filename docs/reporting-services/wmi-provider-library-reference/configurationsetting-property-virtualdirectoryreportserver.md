# <a name="configurationsetting-property---virtualdirectoryreportserver"></a>ConfigurationSetting 屬性 - VirtualDirectoryReportServer
  傳回報表伺服器應用程式的虛擬目錄。 唯讀。  
  
## <a name="syntax"></a>語法  
  
```vb  
Public Dim VirtualDirectoryReportServer As String  
```  
  
```csharp  
public string VirtualDirectoryReportServer;  
```  
  
## <a name="property-values"></a>屬性值  
 包含虛擬目錄的 **string** 。

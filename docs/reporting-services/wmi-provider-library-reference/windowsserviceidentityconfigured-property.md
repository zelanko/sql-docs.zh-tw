# <a name="windowsserviceidentityconfigured-property"></a>WindowsServiceIdentityConfigured 屬性
  傳回上次設定報表伺服器 Windows 服務執行時所使用的識別。 唯讀。  
  
## <a name="syntax"></a>語法  
  
```vb  
Public Dim WindowsServiceIdentityConfigured As String  
```  
  
```csharp  
public string WindowsServiceIdentityConfigured;  
```  
  
## <a name="property-values"></a>屬性值  
 **String** 值，包含傳回上次設定報表伺服器 Windows 服務執行時所使用的識別。

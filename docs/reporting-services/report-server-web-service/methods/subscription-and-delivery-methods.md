# <a name="subscription-and-delivery-methods"></a>訂閱與傳遞方法
  您可以使用這些方法來建立及管理目錄項目的訂閱與傳遞。  
  
|方法|動作|  
|------------|------------|  
|<xref:ReportService2010.ReportingService2010.CreateDataDrivenSubscription%2A>|建立指定之項目的資料驅動訂閱。|  
|<xref:ReportService2010.ReportingService2010.GetDataDrivenSubscriptionProperties%2A>|傳回資料驅動訂閱的屬性。|  
|<xref:ReportService2010.ReportingService2010.CreateSubscription%2A>|在報表伺服器資料庫或 SharePoint 文件庫中，建立指定之項目的訂閱。|  
|<xref:ReportService2010.ReportingService2010.DeleteSubscription%2A>|從報表伺服器資料庫刪除訂閱。|  
|<xref:ReportService2010.ReportingService2010.GetSubscriptionProperties%2A>|傳回訂閱的屬性。|  
|<xref:ReportService2010.ReportingService2010.ListMySubscriptions%2A>|擷取已由報表伺服器或給定目錄項目之 SharePoint 網站的目前使用者所建立的訂閱清單。|  
|<xref:ReportService2010.ReportingService2010.ListSubscriptions%2A>|擷取已為給定項目建立的訂閱清單。|  
|<xref:ReportService2010.ReportingService2010.PrepareQuery%2A>|傳回資料集，其中包含由資料驅動訂閱之傳遞查詢所擷取的欄位。|  
|<xref:ReportService2010.ReportingService2010.SetDataDrivenSubscriptionProperties%2A>|設定資料驅動訂閱的屬性值。|  
|<xref:ReportService2010.ReportingService2010.SetSubscriptionProperties%2A>|設定訂閱的屬性值。|
